## 一、目的

奇門遁甲用於分析賽事時空的能量分布，判斷每匹馬的落宮吉凶、八門八神組合、天干格局等，是 TJSCODE 七術合參的核心流派之一。

Ver.2.7 將奇門升級為 **陰陽雙盤**：
- **道家陰盤**（原 Ver.2.5 內容）：精確至刻，暗干隱干，適合判斷細微伏兵信號。
- **傳統陽盤**（新增）：節氣定局，值符值使，適合判斷宏觀步速與大勢。

兩盤獨立排盤、獨立解讀，在投票制中各自貢獻一票，並在步速預測中協同。

## 二、道家陰盤排盤（強制）

### 2.1 排盤步驟（十步強制，不可跳步）

| 步驟 | 內容 | 備註 |
|------|------|------|
| 1 | 定局數 | 以開跑時間（精確至刻）計算陽遁/陰遁局數：陽遁（冬至後）或陰遁（夏至後），局數 = (時干支數 + 日干支數) mod 9，餘0為9 |
| 2 | 排地盤 | 按洛書九宮順序，將戊己庚辛壬癸丁丙乙對應局數起頭，順時針排列 |
| 3 | 排天盤 | 值符星移時干宮，其餘星按洛書順序順時針旋轉，星移帶原奇（天盤干 = 原宮位地盤干） |
| 4 | 排八門 | 值使門按時干支順序移宮，其餘門按洛書順序順時針排列（陽遁順行，陰遁逆行） |
| 5 | 排九星 | 依天盤值符星定位，其餘星按洛書順序排列 |
| 6 | 排八神 | 值符→螣蛇→太陰→六合→白虎→玄武→九地→九天，陽遁順時針，陰遁逆時針 |
| 7 | 標註暗干 | 星移帶原奇（天盤星原本所在宮位的地盤干） |
| 8 | 標註隱干 | 時干加值使門，其餘門按洛書順序，陽遁順行，陰遁逆行 |
| 9 | 檢查四害 | 擊刑（地支相刑）、入墓（天干入墓庫）、門迫（門剋宮）、空亡（旬空） |
| 10 | 十二長生 | 以地盤天干在落宮的十二長生狀態（長生、沐浴、冠帶、臨官、帝旺、衰、病、死、墓、絕、胎、養） |

### 2.2 用神鎖定

| 投注類型 | 用神 |
|----------|------|
| 獨贏 | 值符、開門、傷門所在宮位 |
| 位置 | 生門、休門所在宮位 |
| 連贏 | 六合所在宮位 |
| 三重彩/四重彩 | 太陰、九天所在宮位 |

  - 太白入熒（庚+丙）
  - 青龍逃走（乙+辛）
  - 白虎猖狂（辛+乙）
  - 青龍返首（戊+丙）
  - 飛鳥跌穴（丙+戊）
  - 日奇入墓（乙+己）
  - 地戶逢鬼（己+己）
  - 青龍入水（戊+壬）
  - 小蛇化龍（壬+戊）
  - 天網四張（癸+辛）
  - 等等

三、傳統陽盤排盤（輔助，強制執行）

3.1 排盤規則

· 節氣定局：以賽事日期所在的節氣三元（上、中、下元）定陽遁或陰遁局數。
· 排盤簡化：只需輸出值符星、值使門、時干所在宮位、關鍵用神宮位（生門、開門、死門等）。
· 重點關注：
  · 值使門（驚門/死門 → 快步速，休門/生門 → 慢步速）
  · 生門+天芮（吉）
  · 開門+天心（吉）
  · 死門+白虎（凶）

3.2 與陰盤的矛盾處理

· 當陰陽兩盤對同一馬匹的吉凶判斷矛盾時，優先採信陰盤（因陰盤在實戰中驗證更多），但需在投票表中標註「陰陽矛盾」，並記錄矛盾點（DP-237）。
· 矛盾案例：R6 捷威（陰盤驚門+玄武吉，陽盤死門+白虎凶）→ 快步速下以陰盤為準（DP-244）。

3.3 輸出格式（陽盤）

```markdown
## 奇門陽盤排盤（傳統節氣遁甲）
- 局數：陽遁 X 局（節氣：XX，元：上/中/下）
- 值符星：[星名] 落 [宮位]
- 值使門：[門名] 落 [宮位]
- 時干：[天干] 落 [宮位]
- 關鍵用神：
  - 生門：落 [宮位]，吉凶 [吉/平/凶]
  - 開門：落 [宮位]，吉凶 [吉/平/凶]
  - 死門：落 [宮位]，吉凶 [吉/平/凶]
```

四、陰陽雙盤的投票制應用

在階段 3.5（多術數投票制）中：

維度 來源 投票標準
奇門陰盤 陰盤落宮、八門、八神、格局 吉（+1）/ 平（0）/ 凶（-1）
奇門陽盤 陽盤用神吉凶 吉（+1）/ 平（0）/ 凶（-1）

· 兩盤獨立投票，互不干擾。
· 若矛盾且無法判定，以陰盤為準。

---

五、與步速預測的聯動

在階段 2.5（步速預測）中，奇門法的依據為：

· 陰盤值使門（驚門/死門落兌宮 → 快步速；休門/生門落坎/艮 → 慢步速）
· 陽盤值使門（同上）
· 兩盤結論一致時權重加倍；矛盾時以陰盤為準。

詳見 sop/sop_pace.md。
