## 一、場地概覽

沙田（Sha Tin）為香港新界的主要賽道，擁有 **草地** 及 **全天候泥地** 兩條跑道，是香港規模最大的賽馬場地。草地賽道呈 **逆時針方向**，賽道寬闊，彎位較跑馬地從容。

| 特性 | 草地跑道 | 泥地跑道 |
|------|----------|----------|
| 方向 | 逆時針（左轉） | 逆時針（左轉） |
| 賽道總長 | 約 1900 米 | 約 1600 米 |
| 直路長度 | 約 450 米 | 約 350 米 |
| 路面闊度 | 約 27-30 米（寬闊） | 約 22-25 米 |
| 彎位數量 | 兩個（第一彎、第二彎） | 兩個 |
| 場地類型 | 天然草地 | 沙泥混合（All-Weather Track） |
| 排水系統 | 極佳 | 受天氣影響較小 |

## 二、草地跑道特性

### 2.1 草地基調

沙田草地賽道寬闊平緩，彎位從容，**公平性較跑馬地高**。檔位偏差存在但不如跑馬地極端。整體而言，沙田草地是一條 **實力先決** 的賽道，馬匹質素比檔位更重要。

### 2.2 草地檔位偏差

| 路程 | 最佳檔位 | 不利檔位 | 說明 |
|------|----------|----------|------|
| 1000 米（直路） | 任何 | 無 | 純直路賽，無彎位，檔位幾乎無影響 |
| 1200 米 | 3-8 檔 | 1 檔、13-14 檔 | 內檔易被困，外檔轉彎需繞大圈 |
| 1400 米 | 2-8 檔 | 12-14 檔 | 中內檔可守好位，外檔耗力 |
| 1600 米 | 2-9 檔 | 1 檔、13-14 檔 | 一檔可能被困於內欄，需騎師處理得宜 |
| 1800 米 | 3-10 檔 | 1 檔、13-14 檔 | 中長途，檔位影響下降 |
| 2000 米 | 任何 | 極外檔（13-14） | 長途賽，檔位影響最小 |
| 2200-2400 米 | 任何 | 極外檔（13-14） | 超長途，以實力和氣量分勝負 |

### 2.3 草地檔位玄學對應

沙田草地寬闊，**開門、生門** 落宮之馬在此場地最具優勢：

| 八門 | 對應檔位傾向 | 原因 |
|------|-------------|------|
| 開門 | 中檔（5-10 檔） | 暢通無阻，可靈活取位 |
| 生門 | 中內檔（2-7 檔） | 生機勃勃，轉彎後發力空間足 |
| 休門 | 中檔（4-9 檔） | 養精蓄銳，適合後追 |
| 傷門 | 中內檔（2-6 檔） | 利競爭搶位，前領可用 |
| 杜門 | 任何檔位 | 沙田寬闊，隱藏戰術空間大 |
| 死門 | 極外檔（12-14 檔） | 草地外檔雖不如泥地致命，但仍需留神 |

## 三、泥地跑道特性

### 3.1 泥地基調

沙田泥地為全天候跑道，沙泥混合，**前排優勢極大**。因泥地踢起的沙泥會影響後方馬匹視線及呼吸，領放或居前馬匹有明顯優勢。後追馬在泥地面對較大困難。

**⚠️ 補丁 DP-001 強制執行**：泥地賽事中，檔位 ≥ 9 檔的馬匹，即使卦象極強，亦不可作為核心獨贏馬。詳見 `rules/patches.md`。

### 3.2 泥地檔位偏差

| 路程 | 最佳檔位 | 不利檔位 | 說明 |
|------|----------|----------|------|
| 1200 米 | 1-4 檔 | 9-12 檔 | 短途泥地，內檔前領優勢極大 |
| 1650 米 | 1-5 檔 | 10-12 檔 | 中距離，前排馬可控局 |
| 1800 米 | 2-6 檔 | 10-12 檔 | 泥地長途較少見，仍以內檔優先 |

### 3.3 泥地玄學對應

泥地賽事中，奇門 **坎宮（水）** 的馬匹需要特別解讀：

| 奇門組合 | 泥地含義 |
|----------|----------|
| 坎宮 + 休門 | 泥地大利。水入泥地，如魚得水 |
| 坎宮 + 天蓬星 | 冒險突襲，泥地伏兵之兆 |
| 坎宮 + 死門 | ⚠️ 水入死地，泥地高危，強制排除 |
| 離宮（火）+ 泥地 | 火被土洩，前領馬在泥地未必能持久 |

## 四、直路賽（沙田 1000 米）特別分析

沙田 1000 米為全港唯一純直路賽道，無彎位，特殊性極高。

### 4.1 直路賽特性

- 無彎位，**純粹比拼速度與爆發力**。
- 檔位幾乎無影響（無需轉彎慳位）。
- 領放馬在直路賽有優勢，但末段 100 米才是決勝區。

### 4.2 直路賽玄學對應

| 玄學指標 | 直路賽含義 |
|----------|-----------|
| 梅花體卦為震☳（木） | 爆發力強，利直路賽 |
| 梅花體卦為離☲（火） | 前領明麗，領放有利 |
| 奇門天沖星 + 星歸本位 | 衝擊力加倍，短途直路最強信號 |
| 奇門傷門 + 落震三宮 | 競爭力強，直路對抗佔優 |
| 六爻子孫爻旺相 + 二爻動 | 前段爆發，直路賽加分 |

## 五、場地狀況影響

### 5.1 草地狀況

| 狀況 | 影響 |
|------|------|
| 好地（Good） | 標準狀況，公平 |
| 好至快地（Good to Firm） | 場地偏快，前領馬優勢增加 |
| 好至黏地（Good to Yielding） | 水分稍多，後追馬機會增加 |
| 軟地（Soft） | 考驗耐力，長途氣量馬有利 |
| 大爛地（Heavy） | 極端情況，香港少見 |

### 5.2 泥地狀況

| 狀況 | 影響 |
|------|------|
| 正常（Fast） | 泥地標準，前排優勢明顯 |
| 濕泥（Wet Fast） | 水分稍多，泥濘影響減低，後追稍有利 |
| 慢泥（Muddy） | 極度泥濘，前排優勢被削弱，耐力馬有利 |

## 六、玄學場地關聯總結

### 6.1 梅花易數與沙田

| 體卦 | 草地表現 | 泥地表現 |
|------|----------|----------|
| 震☳（木） | 短途草地爆發力強 | 泥地平平，不特別有利 |
| 離☲（火） | 前領草地表現佳 | 泥地前領較耗力，需留意 |
| 坎☵（水） | 草地一般 | **泥地有加分**，水路相親 |
| 坤☷（土） | 長途草地耐力足 | 泥地耐力馬亦可 |
| 乾☰（金） | 草地前領剛健 | 泥地前領需內檔配合 |
| 兌☱（金） | 草地短途靈活 | 泥地一般 |
| 巽☴（木） | 草地後追順入 | 泥地後追難度大 |
| 艮☶（土） | 草地長途穩健 | 泥地長途穩健 |

### 6.2 奇門遁甲與沙田

| 奇門組合 | 沙田含義 |
|----------|----------|
| 開門 + 天心星 | 戰術執行完美，沙田寬闊賽道可充分發揮 |
| 生門 + 天任星 | 穩步後追，沙田長直路有足夠空間追上 |
| 值符 + 離九宮 | 天選之馬，沙田大賽日常見此組合勝出 |

## 七、部署建議摘要

| 賽事類型 | 核心策略 |
|----------|----------|
| 沙田草地 1000 米 | 純速度，體卦震/離優先，檔位不重要 |
| 沙田草地 1200 米 | 中內檔前領或中檔後追皆可 |
| 沙田草地 1400-1600 米 | 最公平路程，以卦象及能量氣場為重 |
| 沙田草地 1800-2000 米 | 中長途，檔位影響小，體卦坤/艮有利 |
| 沙田草地 2200 米+ | 純氣量對決，騎師能量指數及馬匹耐力卦為重 |
| 沙田泥地 1200 米 | 內檔前領絕對優先，DP-001 強制執行 |
| 沙田泥地 1650 米 | 前排馬為膽，外檔馬降為配腳 |
| 沙田大賽日 | 值符/青龍等吉神加持之馬優先 |
