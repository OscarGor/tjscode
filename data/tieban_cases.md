## 一、總綱

本檔案記錄 TJSCODE 在實戰中對六爻預測學（增刪卜易五層SOP）名次區間判定的**驗證結果**與**規律發現**。核心驗證公式為：

| 世爻 + 子孫狀態 | 名次區間 |
|---------------|----------|
| 世旺 + 子旺 | 1-3名 |
| 世旺 + 子弱 | 2-4名 |
| 世弱 + 子旺 | 3-5名 |
| 世弱 + 子弱 | 6名或以後 |

**核心原則**：六爻名次區間為輔助判定工具，不可單獨用於排除，需配合梅花卦象及奇門能量進行綜合判斷。

## 三、規律總結

### 3.1 六爻框架被超越的條件

當以下信號出現時，六爻名次區間可被超越：

| 超越方向 | 條件 | 判例 |
|----------|------|------|
| **向上超越**（實名次優於預測） | 梅花卦象極強（UR/SAR級）+ 奇門格局特殊（值符/青龍入海/生門得地） | 勁爽（3-5→2）、金快飛飛（3-5→1） |
| **向上超越** | 豐卦DP-013加持 | 勁爽（3-5→2） |
| **向上超越** | 同宮高評級馬共享能量（DP-018） | 勁爽（與神駒馬靈同宮） |
| **向下超越**（實名次差於預測） | 衰+天芮雙重打擊（DP-033） | 競駿傳奇（3-5→9） |
| **向下超越** | 出閘失誤/碰撞（DP-042） | 爽飆王子（2-4→8） |
| **向下超越** | 梅花凶卦拖累（困/剝/明夷） | 競駿天下（3-5→11） |

### 3.2 六爻最適用的場景

- **正路賽事**（劇本為正路主導）：六爻準確率較高。
- **大熱門驗證**：六爻可輔助判斷大熱門是否可信（世旺+子旺=可信；世弱+子弱=高危）。
- **排除確認**：世弱+子弱+名次區間6+，作為排除的輔助舉證。

### 3.3 六爻不適用的場景

- **冷門活躍劇本**：伏兵信號強烈時，六爻可能低估冷門馬。
- **特殊卦象馬匹**：豐卦、同人卦、益卦等特殊卦象馬匹，以梅花為尊。
