## 一、總綱

本 SOP 為道家陰盤奇門遁甲應用於賽馬預測的**強制操作標準**。每一匹參賽馬，不論賠率冷熱、不論檔位優劣，**必須完整通過以下十步分析**，不得跳步、不得合併、不得因個人經驗而簡化。

**核心原則**：一馬一盤，逐匹落宮。每一步皆有紀錄，每一步皆可稽查。

**與陽盤的核心差異**：
- 定局使用「農曆月日時和÷9取餘」，非節氣
- 天盤九星順時針旋轉（非飛星）
- 必須推算暗干（星移帶原）與隱干（時加使門）
- 寄宮產生多重天干格局，必須全部列出


## 二、十步框架總覽

| 6 | 天加地格局（強制輸出格局名稱） | 所有天干組合逐一分析，有寄干須分析所有組合（含暗干+地盤干） |

## 三、排盤前置（每場賽事僅執行一次）

### 3.1 定局
- 農曆月份數 + 農曆日期數 + 時辰地支序數，和 ÷9 取餘數（餘0為9局）
- 冬至後夏至前 → 陽遁；夏至後冬至前 → 陰遁

### 3.2 地盤三奇六儀
- 戊起於局數宮位，陽遁順時針排戊己庚辛壬癸丁丙乙，陰遁逆時針

### 3.3 值符、值使
- 旬首所在宮位 → 值符（該宮原九星）、值使（該宮原八門）

### 3.4 天盤九星
- 值符星移至時干所在宮位，其餘九星順時針旋轉（天禽隨天芮）

### 3.5 人盤八門
- 值使門從旬首宮位順數（陽遁）或逆數（陰遁）至時辰間隔數，落宮後其餘八門順時針旋轉

### 3.6 神盤八神
- 值符神寫在天盤值符星落宮，陽遁順時針排（值符、螣蛇、太陰、六合、白虎、玄武、九地、九天），陰遁逆時針

### 3.7 暗干（星移帶原）
- 每個宮位的暗干 = 該宮天盤星原在地盤的奇儀
- 值符星暗干特別標註

### 3.8 隱干（時加使門）
- 時干寫在值使門所在宮位，陽遁順時針、陰遁逆時針將其餘天干佈入八門之下

### 步驟一：落宮定位（檔位對應九宮及寄宮）

**目的**：將馬匹檔位轉化為奇門九宮位置，作為全部分析的地理基礎。

**操作規則**：
| 檔位 | 落宮 | 備註 |
|------|------|------|
| 1 | 坎一宮 | 直接對應 |
| 2 | 坤二宮 | 直接對應 |
| 3 | 震三宮 | 直接對應 |
| 4 | 巽四宮 | 直接對應 |
| 5 | 中五宮 | 陽遁寄坤二，陰遁寄艮八 |
| 6 | 乾六宮 | 直接對應 |
| 7 | 兌七宮 | 直接對應 |
| 8 | 艮八宮 | 直接對應 |
| 9 | 離九宮 | 直接對應 |
| 10 | 寄中五（同5） | 陽遁寄坤二，陰遁寄艮八 |
| 11 | 寄乾六 | 同乾六宮 |
| 12 | 寄兌七 | 同兌七宮 |
| 13 | 寄艮八 | 同艮八宮 |
| 14 | 寄離九 | 同離九宮 |

**特別注意**：
- 檔位 10-14 必須標註「寄宮」，並註明所寄之宮。
- 寄宮馬匹與本宮馬匹共享該宮所有信息（八門、八神、九星、地盤干），但天盤干、暗干、隱干可能因天盤星不同而有差異。

**輸出範例**：
> 馬號 5 檔位 5 → 中五宮（陽遁寄坤二）
> 馬號 13 檔位 13 → 寄艮八宮（本宮艮八）

### 步驟二：四害檢查

**目的**：識別落宮是否受四害影響，四害為奇門大凶之象。

**四害定義**：

| 四害 | 定義 | 賽馬含義 |
|------|------|----------|
| 擊刑 | 天干受地盤相刑 | 騎師與場地不合，易犯規失位 |
| 干伏吟 | 天盤干與地盤干相同 | 馬匹狀態停滯，難以突破 |
| 門伏吟 | 八門落本宮 | 戰術無法施展，困守原地 |
| 星伏吟 | 九星落本宮 | 天時不利，能量被封鎖 |

**操作**：
1. 逐一檢查四害是否存在。
2. 若存在任何一害，須明確標示並記錄其賽馬含義。
3. 四害齊全者，該馬定為「凶格」，須特別標註。

**輸出範例**：
> 震三宮 - 四害檢查：
> - 擊刑：無
> - 干伏吟：無
> - 門伏吟：有（傷門落震宮）
> - 星伏吟：無
> 結論：有門伏吟，戰術施展受限。

### 步驟三：十二長生狀態

**目的**：判斷天盤干在落宮的旺衰狀態，決定馬匹當日「生命力」。

**十二長生順序**：
長生 → 沐浴 → 冠帶 → 臨官 → 帝旺 → 衰 → 病 → 死 → 墓 → 絕 → 胎 → 養

**五行長生位**：

| 天干 | 長生宮 | 帝旺宮 | 墓庫 |
|------|--------|--------|------|
| 甲 | 亥（乾六） | 卯（震三） | 未（坤二） |
| 乙 | 午（離九） | 寅（艮八） | 戌（乾六） |
| 丙 | 寅（艮八） | 午（離九） | 戌（乾六） |
| 丁 | 酉（兌七） | 巳（巽四） | 丑（艮八） |
| 戊 | 寅（艮八） | 午（離九） | 戌（乾六） |
| 己 | 酉（兌七） | 巳（巽四） | 丑（艮八） |
| 庚 | 巳（巽四） | 酉（兌七） | 丑（艮八） |
| 辛 | 子（坎一） | 申（坤二） | 辰（巽四） |
| 壬 | 申（坤二） | 子（坎一） | 辰（巽四） |
| 癸 | 卯（震三） | 亥（乾六） | 未（坤二） |

**操作**：
1. 確定天盤干及寄干。
2. 查表找出該干的長生位。
3. 順數至落宮，判定十二長生階段。
4. 長生/冠帶/臨官/帝旺 = 吉；衰/病/死/墓/絕 = 凶；胎/養 = 平。

**帝旺制化條款（DP-041v3）**：
- 帝旺+落宮五行與月令一致 → 真旺
- 帝旺+不一致 → 虛火
- 得制（外檔/負重/強敵）→ 加分半星
- 得洩（領放/早段用力）→ 加分半星
- 無制無洩 → 降半星
- 再生扶 → 過載降一星

**輸出範例**：
> 天盤干：甲 → 長生在乾六 → 落震三宮 = 帝旺（大吉）
> 月令夏季火 → 木休囚 → 虛火，需制化

### 步驟四：天干組合（完整）

**目的**：列出落宮所有天干，為步驟六的格局分析做準備。

**必須列出的天干**：
1. **天盤干**（來自天盤星原在地盤的奇儀）
2. **地盤干**（該宮地盤奇儀）
3. **寄干**（若落宮為寄宮，則寄宮原宮的天盤干？準確：寄宮本身地盤干不變，但天盤干來自寄宮天盤星的原宮。為避免混亂，寄宮馬匹需同時分析本宮和寄宮的組合，但實務上以寄宮的地盤干為準。按陰盤規則：檔位寄宮時，落宮為所寄之宮，地盤干為該宮地盤干，天盤干為該宮天盤星所帶的原宮奇儀。）
4. **暗干**（該宮天盤星原在地盤的奇儀，已體現在天盤干中？暗干其實就是天盤干。但為明確，可重複標註。）
5. **隱干**（依口訣推算，藏於八門之下）

**簡化要求**：至少必須列出 **天盤干** 和 **地盤干**。若有寄宮，必須同時列出寄宮的地盤干和寄宮的天盤干。

**輸出範例**：
> 離九宮 - 天干組合：
> - 天盤干：丙
> - 地盤干：壬
> - 寄干：無
> - 暗干：丙（與天盤干同）
> - 隱干：乙

---

### 步驟五：地支組合

**目的**：列出落宮所有地支，為步驟六的格局分析做準備。

**操作**：
1. 記錄地盤地支（依宮位固定：坎子、坤未申、震卯、巽辰巳、中無、乾戌亥、兌酉、艮丑寅、離午）。
2. 若天盤干自帶地支（甲子、乙丑等），一併記錄。
3. 若寄干帶入外地支，一併記錄。

**輸出範例**：
> 離九宮 - 地支組合：
> - 地盤地支：午
> - 天干自帶：丙午
> - 寄干地支：無

---

### 步驟六：天加地格局分析（強制輸出格局名稱）

**目的**：分析天盤干與地盤干的組合關係，此為奇門判斷吉凶的核心。

**操作規則**：
1. 以「天盤干 + 地盤干」形式組合。
2. 若有寄宮，須分析「天盤干（寄宮）+ 地盤干（寄宮）」以及「天盤干（本宮）+ 地盤干（寄宮）」？實務簡化：以馬匹實際落宮（含寄宮）的地盤干為準，天盤干為該宮天盤星所帶。
3. 必須輸出標準格局名稱（如「青龍逃走」、「太白入熒」、「小蛇化龍」等），不可只寫組合。
4. 若有暗干或隱干與地盤干形成第二組格局，也須列出。

**常見格局速查表（陰盤常用）**：

| 格局 | 組合 | 吉凶 | 賽馬應用 |
|------|------|------|----------|
| 青龍返首 | 戊+丙 | 大吉 | 求財大旺，大熱門可信 |
| 飛鳥跌穴 | 丙+戊 | 大吉 | 機會主動送上門 |
| 天乙會合 | 癸+戊 | 吉（利客） | 暗中求財，貴人隱於暗處 |
| 太白入熒 | 庚+丙 | 吉（利客） | 後上突襲，冷門伏兵 |
| 熒入太白 | 丙+庚 | 吉（利主） | 防守反擊 |
| 青龍逃走 | 乙+辛 | 大凶 | 破財撤資，不可做膽（可被DP-097/107壓制） |
| 白虎猖狂 | 辛+乙 | 大凶 | 車禍血光，大熱倒灶 |
| 朱雀投江 | 丁+癸 | 大凶 | 文書遺失，合約失敗 |
| 螣蛇夭矯 | 癸+丁 | 凶 | 口舌官司，驚恐怪異 |
| 地戶逢鬼 | 己+己 | 凶 | 暗中消耗，百事不遂 |
| 日奇入墓 | 乙+己 | 平偏凶 | 希望被遮蔽，需吉門化解 |
| **小蛇化龍** | 壬+戊 | 中吉 | 轉變提升，冷門變熱 |
| **青龍入水** | 戊+壬 | 中平 | 財被耗，但可守 |
| **單癸空亡** | 僅癸無天盤 | 中平 | 初出神秘，需杜門九天配合 |

**輸出範例**：
> 坎一宮 - 格局分析：
> - 天盤癸 + 地盤戊 = 天乙會合（吉）。暗中有貴人，利留後衝刺。
> - 無寄干。

> 中五宮寄坤二（檔位5）：
> - 本宮（中五）：天盤乙 + 地盤己 = 日奇入墓（平偏凶）。希望被遮蔽。
> - 寄宮（坤二）：天盤庚 + 地盤丙 = 太白入熒（吉）。主動攻擊，利冷門突襲。
> - 結論：多重格局，凶中藏吉。

### 步驟七：八門分析

**目的**：判斷八門吉凶及其對賽馬的影響。

**八門屬性**：

| 八門 | 吉凶 | 賽馬含義 |
|------|------|----------|
| 開門 | 大吉 | 開路先鋒，利領放、突破 |
| 休門 | 吉 | 養精蓄銳，利留前鬥後 |
| 生門 | 大吉 | 生機勃勃，利後上追趕 |
| 傷門 | 中（利競爭） | 主動出擊，利搶位、對抗 |
| 杜門 | 中（利隱藏） | 隱藏實力，利伏兵突襲 |
| 景門 | 中平 | 表面風光，宜睇定形勢 |
| 死門 | 大凶 | 前路不通，阻滯重重 |
| 驚門 | 凶 | 心神不寧，易犯錯失機 |

**門迫吉門降級（DP-075）**：
- 吉門（開、休、生）落入門迫宮位 → 不可做獨贏膽
- 開門門迫（落震三、巽四）→ 事業受阻
- 休門門迫（落離九）→ 懶散消極
- 生門門迫（落坎一）→ 財路斷絕

**操作**：
1. 記錄落宮八門。
2. 判定吉凶，並寫出賽馬含義。
3. 結合馬匹跑法特性：前領馬遇開門加分，後追馬遇生門加分。

**輸出範例**：
> 震三宮 - 八門：傷門。
> 含義：利競爭、利搶位、利主動出擊。符合該馬前領跑法，加分。

---

### 步驟八：八門+天盤干組合分析

**目的**：將八門與天盤干組合，細化判斷。

**操作**：
1. 將落宮八門與所有天盤干（含寄干）配對。
2. 分析組合含義，例如：
   - 開門 + 戊 = 開門逢財，大利前領
   - 生門 + 丙 = 生生不息，後追強勁
   - 死門 + 庚 = 絕路逢阻，凶上加凶
   - 休門 + 太陰（八神） + 內檔 → 內檔相沖降級（DP-027v2）
3. 有寄干時須分析多組。

**輸出範例**：
> 震三宮 - 八門+天盤組合：
> - 傷門 + 甲：木氣相疊，競爭力倍增（吉）
> - 無寄干。

---

### 步驟九：九星分析

**目的**：判斷天時能量對馬匹的加持或壓制。

**九星屬性**：

| 九星 | 五行 | 吉凶 | 賽馬含義 |
|------|------|------|----------|
| 天蓬 | 水 | 凶 | 冒進易失，利大膽突襲 |
| 天任 | 土 | 吉 | 穩健可靠，利均速跑法 |
| 天沖 | 木 | 中 | 衝擊力強，利短途爆發 |
| 天輔 | 木 | 吉 | 輔助得宜，利團隊配合 |
| 天英 | 火 | 中 | 光芒畢露，利焦點馬匹 |
| 天芮 | 土 | 凶 | 狀態不佳，宜避之（DP-033衰+天芮） |
| 天柱 | 金 | 凶 | 阻力重重，難以發揮 |
| 天心 | 金 | 吉 | 策劃得當，利戰術執行 |
| 天禽 | 土 | 大吉 | 居中調度，萬事皆宜 |

**星歸本位特別條款**：
- 九星落本宮 = 「星歸本位」，能量加倍。
- 例如：天沖星落震三宮（木歸木位），衝擊力倍增，短途馬大利。

**操作**：
1. 記錄落宮九星。
2. 判定吉凶及賽馬含義。
3. 檢查是否星歸本位，若是則特別標註並加重計分。

**輸出範例**：
> 震三宮 - 九星：天沖星。
> 含義：衝擊力強，利短途爆發。
> ⚠️ 星歸本位！天沖（木）落震三（木），能量加倍。評級：大吉。

### 步驟十：八神分析

**目的**：判斷神秘能量對馬匹的影響，八神為奇門最高層次能量。

**八神屬性**：

| 八神 | 吉凶 | 賽馬含義 |
|------|------|----------|
| 值符 | 大吉 | 天時所歸，貴人相助，頭馬之選 |

**值符救應上限（DP-084）**：
- 奇門凶格（死門+壬己、傷門+伏宮等）逢值符
- 最多升一級（如 C→R，R→AR）
- 不可做獨贏膽，不可高於 SR

**操作**：
1. 記錄落宮八神。
2. 判定吉凶及賽馬含義。
3. 值符落宮之馬須特別標註，為「天選之馬」重要參考，但須遵守 DP-084 上限。

**輸出範例**：
> 震三宮 - 八神：值符。
> 含義：天時所歸，該駒獲神秘加持，列為頭馬候選。評級：大吉。

八、強制執行條款

1. 覆蓋率 100%：每場賽事全部參賽馬匹，必須逐匹完成十步分析，稽查部負責核實。
2. 寄干雙重分析：凡涉及檔位5或10-14的寄宮，步驟四、六、八必須分析兩組天干組合（本宮+寄宮）。
3. 格局名稱強制輸出：步驟六不得只寫「丙+戊」，必須寫「丙+戊 = 飛鳥跌穴」。
4. 星歸本位標註：步驟九必須明確標示是否星歸本位，漏標視為違規。
5. 值符強制標註：步驟十若出現值符，須在綜合評級中特別列明，並遵守 DP-084 上限。
6. 特殊格局強制標註：出現天乙會合、太白入熒、青龍逃走、小蛇化龍、青龍入水等格局，必須標註對應補丁。
7. 跳步處分：任何跳步行為，稽查部有權退回整份分析，要求重做。
