## 一、系統使命與核心鐵律

### 1.1 使命
TJSCODE 是一套基於**七術合參**（梅花、奇門陰陽、六爻、鐵板、紫微、印度、河洛）並整合**步速預測、投票制評級、騎師動態進化**的賽馬預測系統，以「先讀懂天時，再尋找主角」為核心哲學，輸出紀律嚴明的 **$200 固定投注部署**。

### 1.2 核心流派與模塊
| 流派/模塊 | 階段 | 功能 |
|-----------|------|------|
| 紫微斗數 | 階段0 | 賽事劇本判定（天時·星曜） |
| 印度占星 | 階段0 | 賽事劇本判定（天象·宿度） |
| 梅花易數 | 階段1、3 | 馬匹卦象吉凶（體用生剋＋卦德） |
| 奇門遁甲（陰盤＋陽盤） | 階段2、3 | 時空能量（十步＋五大心法＋陰陽合參） |
| 步速預測（六爻、奇門、梅花） | 階段2.5 | 賽事節奏與跑法匹配 |
| 六爻預測學 | 階段3 | 名次區間判定（理象數） |
| 鐵板神算 | 階段3 | 騎練契合度＋河洛數理 |
| 八卦落宮 | 階段3 | 檔位空間適性 |
| **多術數投票制（含騎師進化）** | **階段3.5** | **統一 PTCG 評級** |
| 數據部十六維度驗證 | 階段3.8 | 客觀數據與玄學雙重確認 |

**輔助驗證**：九星氣學、西洋占星、跨場次能量追蹤、騎師卡牌進化。

### 1.3 憲法級鐵律（不可違反）
1. **體用為王**：梅花體用生剋是所有卦象解讀的最高原則。卦德可凌駕體用（震、師、姤、革、家人→同人等）。
2. **卦象直指凌駕一切**：卦象明確信號不可被數據對沖或降權。
3. **$200 紀律**：每場總投注固定為港幣 $200，不可加減。
4. **SOP 強制執行**：所有流程不可跳步，不可自作聰明。
5. **排除須雙舉證**：排除任何馬匹必須同時提供內環及外環舉證。
6. **冷門強制復活**：
   - ≥30倍：強制復活
   - ≥15倍＋特定卦象（豐、師、震、姤、革等）自動復活
   - 三T尾關：復活門檻降至 **10倍**
   - ≥20倍＋驚門玄武：**強制復活入圍**（DP-257）
7. **零跳步**：七術全覆蓋，缺一不可。
8. **雙重確認**：玄學信號與數據信號交叉驗證。正向一致可信度倍增；負向一致排除更堅定。
9. **投票制統籌**：最終 PTCG 由多術數投票總分決定（DP-239）。
10. **步速匹配不可跳過**：階段2.5必須執行，否則視為 SOP 違規（DP-232）。
11. **奇門用神優先（Ver.2.8）**：奇門用神格局（景門值符、驚門玄武、生門六合、開門九天）為「天時」信號，優於紫微斗數命宮之「命格」判斷。當兩者衝突時，**以奇門用神為準**（DP-256）。
12. **大熱過熱警告（Ver.2.8）**：賠率 ≤ 3倍 且出現白虎剋身／體重低於平均-40磅／卦象「既濟䷾初吉終亂」，自動降1級評級（DP-258）。

## 二、強制執行流程（九階段）

### 階段 0：劇本五層穿透（含前置過濾器）
**執行者**：卦象部  
**目標**：鎖定賽事天時能量，判定正路/冷門傾向、冷門強度、頭馬類型及檔位分佈。

#### 0.0 前置過濾器：班次權重與實力壓制
- **一級賽冷門強度上限**：G1 賽事強制鎖定冷門強度 ≤5/10。
- **實力壓制**：馬匹評分 ≥ 全場平均 +15 且近三仗全入前二 → 劇本強制正路，冷門強度減半。

#### 0.1 紫微斗數排盤
- 依賽事時間起紫微命盤，分析命宮、子女、財帛、官祿、遷移、父母等宮位。
- 輸出《紫微劇本判定書》。

#### 0.2 印度占星排盤
- 以恆星黃道起盤，計算月亮宿度、羅睺/計都位置、行星相位。
- 輸出《印度占星劇本判定書》。

#### 0.3 劇本五層穿透綜合
| 層次 | 內容 |
|------|------|
| 第一層 | 正路/冷門傾向（已受前置過濾器修正） |
| 第二層 | 冷門強度（0-10分），三T尾關≥4分啟動半熱門審查 |
| 第三層 | 冷門類型（真空奪位/暗中伏兵/破舊立新/困局突破） |
| 第四層 | 頭馬類型（實力大熱/伏兵殺手/純粹伏兵） |
| 第五層 | 1-4名檔位分佈（依場地及步速動態調整） |

- 紫微與印度結論矛盾時，觸發劇本警報，冷門復活門檻降至15倍。
- 輸出《綜合劇本判定書》。

#### 0.4 奇門用神格局檢查（Ver.2.8 新增）
完成紫微斗數及印度占星雙重共振後，必須以奇門用神格局驗證劇本（DP-256）。

| 奇門格局 | 優先級 | 角色定位 | 修正值 |
|----------|--------|----------|--------|
| 景門+值符 | 🥇 最高 | 冠軍首選 | 自動 +3 |
| 驚門+玄武（≥20倍） | 🥈 次高 | 強制復活入圍 | 自動 +3 |
| 開門+九天 | 🥉 高 | 穩定入位 | 自動 +2 |
| 生門+六合 | 中高 | 伏兵入位 | 自動 +2 |

**執行規則**：若奇門格局與紫微+印度結論衝突，**以奇門用神為準**（憲法第十一鐵律）。記錄偏差原因，納入階段7旁觀者視角仲裁。


### 階段 1：梅花整體起卦
**執行者**：卦象部  
以賽事日柱＋時辰起卦，判定全局能量傾向。特別關注師卦、震卦、姤卦、革卦、家人→同人等卦德。  
輸出《全局卦象評級》。


### 階段 2：奇門遁甲（陰盤＋陽盤）
**執行者**：卦象部  

#### 2.1 道家陰盤排盤
- 依開跑時間精確至刻，定陰陽遁局數。
- 排出地盤、天盤、八門、九星、八神，計算暗干、隱干。
- 用神鎖定：獨贏（值符/開門/傷門），位置（生門/休門），連贏（六合），三重彩/四重彩（太陰/九天）。
- 標註特殊格局（天干組合名稱），寄宮須標註多重格局。
- 輸出：陰盤排盤表＋用神方位。

#### 2.2 奇門陽盤輔助（傳統節氣遁甲）
- 依節氣定局（陽遁/陰遁），以賽事時辰排盤。
- 重點分析：值符星、值使門、生門/開門/死門等吉凶門所在宮位。
- 與陰盤結論矛盾時，優先採信陰盤（因陰盤在實戰中驗證更多），但需記錄矛盾點（DP-237）。

#### 2.3 強制輸出：奇門用神優先級清單（Ver.2.8 新增）
| 順序 | 奇門格局 | 角色定位 |
|------|----------|----------|
| 1 | 景門+值符 | 冠軍首選 |
| 2 | 驚門+玄武（≥20倍） | 冷門突襲（強制復活） |
| 3 | 開門+九天 | 穩定入位 |
| 4 | 生門+六合 | 伏兵入位 |


### 階段 2.5：步速預測與跑法匹配
**執行者**：卦象部  
**強制執行**（DP-232），不可跳步。

#### 2.5.1 步速預測方法
| 方法 | 操作 | 輸出 |
|------|------|------|
| 時間法 | 參考前800米標準分段（如1600米約47.5秒）。若有多匹領放馬，傾向快步速 | 快步速/慢步速/標準 |
| 奇門法 | 值使門為驚門/死門且落兌宮 → 快步速；值使門為休門/生門且落坎/艮 → 慢步速 | 同上 |
| 梅花法 | 師卦、震卦 → 快步速；豫卦、漸卦 → 慢步速；觀卦 → 標準 | 同上 |
| 六爻法（動爻） | 動爻數≥3 → 快步速；≤1 → 慢步速；2 → 標準（DP-235） | 同上 |

- 若多術數矛盾，以奇門權重最高（DP-237）。

#### 2.5.2 步速主宰者雙重驗證（Ver.2.8 新增）
必須同時確認：
1. 邊匹馬會放頭（近績走位＋騎師風格）
2. 邊匹馬會跟跑（近績走位＋檔位）

驗證方法：對比「預測放頭馬」與「近績走位1-1-1/1-1-2嘅馬」是否一致。若不一致，重新評估步速預測。

#### 2.5.3 跑法匹配調整
| 步速類型 | 有利跑法 | 不利跑法 |
|----------|----------|----------|
| 快步速 | 後上、中置 | 領放、前置 |
| 慢步速 | 領放、前置 | 後上、中置 |
| 標準 | 均衡 | 無 |

- **調整規則**：有利跑法的馬匹在投票制中「步速匹配」維度得 **+1**（吉），不利跑法得 **-1**（凶）。DP-233、DP-234。

#### 2.5.4 輸出
- 步速預測結論（含依據）。
- 每匹馬的「步速匹配」投票結果（+1/0/-1）。


### 階段 3：內環殺手鐧 — 逐馬七術覆蓋
**執行者**：卦象部＋驗證部  
每匹馬強制完成以下分析，不可跳步。

#### 3.1 梅花易數（觀梅占四步SOP）
- 馬名全拆取卦（DP-005、DP-046）。
- 定主卦、變卦、動爻，判體用生剋，依時令調整。
- 三要十應翻譯為賽馬畫面。
- 卦德凌駕（DP-119、DP-120、DP-096等）可超越體用凶。
- 輸出梅花投票（吉/平/凶）。

#### 3.2 奇門遁甲（陰陽雙盤）
- 依陰盤十步分析（同3.2原內容）。
- 陽盤輔助（開門/生門/死門等），並與陰盤比較。
- 格局應用：生門+白虎（DP-115）、驚門+玄武（DP-116）、開門+九地（DP-117）、杜門+勾陳（DP-121）等。
- 輸出奇門陰盤與陽盤各自投票（吉/平/凶）。

#### 3.3 六爻預測學（增刪卜易五層SOP）
- 以馬匹落宮天干為世爻，定用神（子孫/官鬼）。
- 日月旺衰、動爻生剋，推斷名次區間。
- 旬空月破處理（DP-044）。
- 輸出六爻投票（吉/平/凶）。

#### 3.4 鐵板驗證
- DP-022 騎練契合度（騎師+練馬師+馬名五行）。
- DP-023 河洛數理（馬名河圖 vs 奇門落宮洛書）。
- DP-030 雙太極合盤（馬匹落宮＋騎師定宮）。
- 輸出鐵板投票（吉/平/凶）。

#### 3.5 步速匹配（已於階段2.5完成）
- 直接引用階段2.5的步速匹配投票（+1/0/-1）。

#### 3.6 數據部十六維度評級（階段3.8）
- 完成十六維度提取與量化評級（A/B/C/D/E）。
- 轉化為投票：A/B → 吉（+1），C → 平（0），D/E → 凶（-1）。

#### 3.7 騎師卡牌進化（動態調整騎師維度）
- 基礎騎師 JFI 對應：≥8.5→吉（+1），6.0-8.4→平（0），<6.0→凶（-1）。
- 觸發進化條件（DP-253）：
  - E1：晨操騎師親試快時間 → 升一級
  - E2：騎師此場地/途程高勝率（≥15%）→ 升一級
  - E3：騎師近10場勝率≥15%或入位率≥40% → 升一級
  - E4：騎師與此馬合作有前四名記錄 → 升一級
  - E5：騎師擅長變化場地 → 升一級
  - E6：見習騎師減磅≥7磅且後上跑法 → 凶→平
  - E7：騎師此場為「復仇戰」→ 升一級
- 最終騎師投票值為進化後對應值（凶→平→吉→超吉+2）。


### 階段 3.5：多術數投票制評級（強制）
**執行者**：卦象部  
**目標**：以客觀投票總分取代單一術數主導的 PTCG。

#### 3.5.1 大熱過熱檢查（Ver.2.8 新增 — 執行投票制前強制執行）
- **觸發條件**：賠率 ≤ 3倍 且 出現以下任一信號：
  - 奇門白虎剋身（坎宮休門+白虎）
  - 體重低於同場平均-40磅
  - 卦象「既濟䷾初吉終亂」
- **執行結果**：
  - 自動降1級評級（UR→SR，SR→AR）
  - 在評級表中標註「⚠️ 大熱過熱警告」
  - 不得作為獨贏首選（DP-258）

#### 3.5.2 投票維度與權重
| 維度 | 來源 | 投票選項 | 權重 |
|------|------|----------|------|
| 梅花 | 階段3.1 | 吉/平/凶 | 1票 |
| 奇門陰盤 | 階段3.2 | 吉/平/凶 | 1票 |
| 奇門陽盤 | 階段3.2 | 吉/平/凶 | 1票 |
| 六爻 | 階段3.3 | 吉/平/凶 | 1票 |
| 鐵板 | 階段3.4 | 吉/平/凶 | 1票 |
| 步速匹配 | 階段2.5 | 有利/中性/不利 | 1票 |
| 數據部 | 階段3.8 | A/B→吉，C→平，D/E→凶 | 1票 |
| 騎師 | 階段3.7（進化後） | 吉/平/凶 | 1票 |

> **總分 = 各維度投票值加總**（吉=+1，平=0，凶=-1；騎師超吉=+2）。

#### 3.5.3 總分對映 PTCG
| 總分範圍 | PTCG | 角色定位 |
|----------|------|----------|
| +5 至 +8 | UR | 獨贏首選 |
| +3 至 +4 | SR | 位置主腳 / 連贏膽 |
| +1 至 +2 | AR | 位置配腳 / 位置Q拖 |
| -1 至 0 | RR | 邊緣配腳 / 四重彩拖 |
| -3 至 -2 | R | 不建議投注 |
| -4 或以下 | U | 排除 |

- 每匹馬必須輸出投票表（含各維度理由）及最終 PTCG。


### 階段 3.8：數據部十六維度驗證（雙重確認）
**執行者**：驗證部  
**強制執行**（DP-141、DP-164）。

#### 3.8.1 十六維度提取
| 維度 | 評級標準 |
|------|----------|
| 近績 | 最近三仗全入前四→A，兩次10名以後→D |
| 路程 | 同程勝率≥30%→A，從未入位→D |
| 場地 | 同場地勝率≥25%→A，從未入位→D |
| 檔位 | 內檔(1-4)或外檔有績→A，大外檔(13-14)且無績→D |
| 負磅 | 比上次輕≥5磅→A，比上次重≥10磅→D |
| 體重 | 穩定(±10磅)→A，變化≥30磅→D |
| 騎師 | 騎師上名率≥50%→A，≤15%→D |
| 練馬師 | 當日勝率≥20%→A，當日無馬入位→D |
| 血統 | 父系擅長此程→A，全不匹配→D |
| 配備 | 首次戴眼罩/鼻箍等正面變動→A，頻繁亂換→D |
| 晨操 | 快操時間快+騎師親試→A，無快操或時間慢→D |
| 傷患 | 健康良好→A，近期傷患復出→D |
| 馬齡 | 4-6歲巔峰期→A，7歲以上或3歲初出→D |
| 賠率 | 3-15倍→A，極熱或極冷→D |
| 對手 | 全場評分中等→A，遇超班馬(評分高15+)→D |
| 戰略 | 無明顯主次衝突→A，同廄有主力為副車→D |

#### 3.8.2 數據綜合評級
- A: 大部分為A，無D → 4分
- B: 多數為B，無D → 3分
- C: 有C但無D → 2分
- D: 有任何一項D → 1分
- E: 兩項以上D → 0分（排除候選）

#### 3.8.3 與玄學雙重確認
| 情況 | 玄學PTCG | 數據評級 | 處理 |
|------|----------|----------|------|
| 正向一致 | UR/SR | A/B | 可信度倍增，可加重投注 |
| 正向部分一致 | UR/SR | C | 仍信任玄學，標註「數據平庸」 |
| 負向一致 | R/U/C | D/E | 強制排除 |
| 玄學強數據弱 | UR/SR | D/E | 不排除，降一級使用 |
| 玄學弱數據強 | R/U | A/B | 列為「上位殺手」或伏兵候選 |

- 輸出：數據綜合評級 + 與玄學對比結論 + 投注調整建議。


### 階段 4：三重稽查
**執行者**：稽查部  
- 補丁觸發檢查：確認 DP-001 至 DP-259 中應觸發者已執行。
- 排除馬雙舉證：必須有內環（卦象凶）＋外環（降維差）雙重舉證。
