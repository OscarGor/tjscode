**核心哲學**：先讀懂天時，再尋找主角。數據的盡頭是玄學，玄學的盡頭是劇本。  
**最高心法**：投票定級，步速為綱，騎師可進，陰陽合璧。

## 快速開始

### 1. 閱讀憲法
`CONSTITUTION.md` — 憲法級鐵律，不可違反。**Ver.2.7 憲法鐵律增至十條**，新增雙重確認、投票制統籌、步速匹配強制執行。

### 2. 執行 SOP
`sop/sop_master.md` — 九階段總 SOP，所有賽事預測必須依序執行，**強制新增階段2.5（步速預測）與階段3.5（多術數投票制）**。

### 3. 查閱補丁
`rules/patches.md` — **255 條補丁**（DP-001 至 DP-255），每一條都是用錢換來的教訓。

### 4. 接收賽事
提供賽事資訊（馬名、騎師、練馬師、檔位、賠率、晨操紀錄等），系統將自動執行完整九階段 SOP。

## PTCG 稀有度評級（投票制總分映射）

TJSCODE Ver.2.7 採用 **多術數投票制** 統一評級。每匹馬由以下八個維度投票（吉+1/平0/凶-1）：梅花、奇門陰盤、奇門陽盤、六爻、鐵板、步速匹配、數據部十六維度、騎師（含進化）。總分映射 PTCG：

| 總分範圍 | PTCG | 角色定位 |
|----------|------|----------|
| +5 至 +8 | UR | 獨贏首選 |
| +3 至 +4 | SR | 位置主腳 / 連贏膽 |
| +1 至 +2 | AR | 位置配腳 / 位置Q拖 |
| -1 至 0 | RR | 邊緣配腳 / 四重彩拖 |
| -3 至 -2 | R | 不建議投注 |
| -4 或以下 | U | 排除 |

**實用性標籤**：主戰力、上位殺手、實用配搭、待開發、小丑、凡骨  
**靚卡廢卡機制**：稀有度高 ≠ 實用。判定以投票總分為準，不以單一流派評級為定論。
