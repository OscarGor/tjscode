## 一、總綱

六爻預測學為 TJSCODE **內環第三流派**，專司「即時占問」與「爻象吉凶」。本 SOP 以《增刪卜易》「刪繁就簡，直奔吉凶」為核心精神，純以五行生剋、日月力量為準，分五層遞進，最終鎖定每匹馬的名次區間。

**最高心訣**：
> 日月為提綱，掌管生殺大權。  
> 動爻為玄機，神兆機於動。  
> 世旺子旺為上品，世弱子弱必無功。


## 二、第一層：立太極（定焦點）

### 2.1 定世爻

以馬匹在奇門遁甲中的落宮天干為世爻，代表馬匹自身。

| 落宮 | 地盤天干 | 世爻五行 |
|------|----------|----------|
| 坎一 | 戊 | 土 |
| 坤二 | 壬 | 水 |
| 震三 | 庚 | 金 |
| 巽四 | 辛 | 金 |
| 中五 | 癸 | 水 |
| 乾六 | 丙 | 火 |
| 兌七 | 丁 | 火 |
| 艮八 | 己 | 土 |
| 離九 | 乙 | 木 |

### 2.2 定用神

| 六親 | 代表 | 賽馬含義 |
|------|------|----------|
| **世爻** | 馬匹自身 | 實力狀態 |
| **子孫爻** | 福神 | 入圍能力、末段衝刺力 |
| **妻財爻** | 獎金回報 | 派彩潛力 |
| **官鬼爻** | 壓力障礙 | 競爭阻力、意外風險 |
| **父母爻** | 外在條件 | 場地、騎練支援 |
| **兄弟爻** | 競爭對手 | 同場其他馬匹威脅 |

**用神多現法則**：若卦中多爻動，優先取臨日辰或月建之爻為用神。

### 2.3 定世應

| 爻位 | 代表 | 賽馬含義 |
|------|------|----------|
| **世爻** | 我方（馬匹自身） | 以馬匹落宮天干為世爻 |
| **應爻** | 對方（賽場環境） | 應爻生世則賽場利馬匹發揮 |


## 三、第二層：審天時（定能量底色）

### 3.1 日月生剋判定

以賽事當日的日辰及月建，判定對世爻的生剋關係。

**日辰生剋**：
| 關係 | 判定 | 加權 |
|------|------|------|
| 日辰生世爻 | 得日助 | +1 |
| 世爻臨日辰（同五行） | 臨日 | +1 |
| 日辰剋世爻 | 受日剋 | -1 |
| 日辰洩世爻（世生） | 洩氣 | -0.5 |
| 無關係 | 平 | 0 |

**月建生剋**：
| 關係 | 判定 | 加權 |
|------|------|------|
| 月建生世爻 | 得月助 | +1 |
| 世爻臨月建（同五行） | 臨月 | +1 |
| 月建剋世爻 | 受月剋 | -1 |
| 月建洩世爻（世生） | 洩氣 | -0.5 |
| 無關係 | 平 | 0 |

### 3.2 綜合旺衰判定

| 日月總分 | 世爻狀態 |
|----------|----------|
| ≥ +1.5 | **旺相** |
| +0.5 至 +1.0 | **偏旺** |
| -0.5 至 +0.5 | **中和** |
| -1.0 至 -0.5 | **偏弱** |
| ≤ -1.5 | **衰弱** |

### 3.3 旬空月破判定（DP-044）

| 旬 | 空亡地支 |
|------|----------|
| 甲子旬 | 戌亥空 |
| 甲戌旬 | 申酉空 |
| 甲申旬 | 午未空 |
| 甲午旬 | 辰巳空 |
| 甲辰旬 | 寅卯空 |
| 甲寅旬 | 子丑空 |

**判定規則**：
- 落宮逢旬空 → 自動降一級稀有度。
- 落宮逢月破（與月建相沖）→ 自動降一級稀有度。
- 旬空+月破同現 → 強制排除。
- 旬空之爻臨日辰 → 出空，不降級。
- 旬空+「胎」「養」長生 → 潛伏蓄力型，降級減半。


## 四、第三層：辨動靜（看事情玄機）

### 4.1 梅花動爻取玄機

以梅花易數起卦時的動爻為「神兆機於動」。

| 動爻關係 | 六親 | 判定 | 加權 |
|----------|------|------|------|
| 動爻生世爻 | 原神 | 外力相助 | +1 |
| 動爻剋世爻 | 忌神 | 外力打壓 | -1 |
| 動爻與世爻比和 | 比助 | 助力 | +0.5 |
| 世爻生動爻 | 洩神 | 消耗 | -0.5 |

### 4.2 變卦回頭判定

| 變卦關係 | 判定 | 加權 |
|----------|------|------|
| 變卦回頭生動爻 | 原神增強 | +0.5 |
| 變卦回頭剋動爻 | 忌神增強 | -0.5 |
| 變卦與動爻比和 | 平穩 | 0 |

### 4.3 子孫爻旺衰判定

子孫爻代表入圍能力，其旺衰取決於動爻及日月對其五行之生剋：

| 條件 | 子孫狀態 |
|------|----------|
| 子孫爻得日月生或臨日月 | **旺** |
| 子孫爻受日月剋洩 | **弱** |
| 子孫爻受動爻生 | **旺** |
| 子孫爻受動爻剋 | **弱** |


## 五、第四層：評吉凶（下終極結論）

### 5.1 綜合吉凶判定矩陣

| 世爻狀態 | 動爻關係 | 子孫爻狀態 | 吉凶 | 說明 |
|----------|----------|-----------|------|------|
| 旺相 | 原神生 | 旺 | **大吉** | 實力強＋外力助＋入圍力強 |
| 旺相 | 原神生 | 弱 | **吉** | 實力強＋外力助，入圍力稍弱 |
| 旺相 | 忌神剋 | 旺 | **受阻** | 實力強但受打壓，仍可入圍 |
| 旺相 | 忌神剋 | 弱 | **受阻偏弱** | 實力強但雙重打壓 |
| 偏旺 | 原神生 | 旺 | **吉** | 實力中上＋外力助 |
| 偏旺 | 忌神剋 | 弱 | **中平** | 外力打壓抵消實力 |
| 中和 | 原神生 | 旺 | **中上** | 外力補足實力 |
| 中和 | 忌神剋 | 弱 | **中下** | 外力打壓 |
| 偏弱 | 原神生 | 旺 | **克處逢生** | 先凶後吉，逆境突圍 |
| 偏弱 | 忌神剋 | 弱 | **凶** | 雙重打擊 |
| 衰弱 | 原神生 | 旺 | **弱中帶旺** | 實力弱但有救援 |
| 衰弱 | 忌神剋 | 弱 | **大凶** | 三重打擊，強制排除 |

### 5.2 特殊情況

| 情況 | 判定 |
|------|------|
| **克處逢生** | 世爻衰弱但有原神動來生 → 名次可能受困後回升，可入圍 |
| **貪合忘克** | 忌神被日辰或月建合住 → 擠阻可避開，逃過一劫 |
| **旬空** | 末段追近，需留意後上跑法 |
| **月破** | 本場無望，下仗回升 |

### 6.2 旬空月破對名次的影響
