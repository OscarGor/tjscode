## 一、總綱

稽查執行SOP為 TJSCODE **階段4（三重稽查）** 及 **階段6（SOP三級審核）** 的強制操作手冊。其核心任務是確保階段3的內環分析無跳步、無主觀偏見、無補丁遺漏。任何未通過稽查的馬匹分析，必須退回階段3重做。

**稽查鐵律**：
> 跳步即違規，漏丁即失職。  
> 體用為王，不可曲解。  
> 排除須雙舉，冷門必復活。

## 二、稽查A：奇門遁甲SOP合規審查

**執行者**：Tim  
**審查對象**：階段3.2 奇門遁甲十步分析  
**審查標準**：依 `sop/sop_qimen.md` 及 `wisdom/qimen.md`

### 2.1 十步覆蓋率審查

| 檢查項 | 標準 | 不合格判定 |
|--------|------|-----------|
| 落宮定位 | 每匹馬必須明確標示落宮宮位 | 漏標或錯標 |
| 四害檢查 | 擊刑、入墓、門迫、空亡逐一標示 | 任一項漏檢 |
| 十二長生 | 地盤天干在落宮的長生狀態必須標示 | 漏標 |
| 天干組合 | 天盤+地盤組合及格局名必須標示 | 漏標 |
| 四位一體 | 天(九星)、人(八門)、地(九宮)、神(八神)必須一體化解讀 | 僅列舉符號而無整合 |
| 旺衰動靜 | 月令旺衰及八門得地與否必須判定 | 漏判 |
| 能量阻塞拆解 | 擊刑/入墓/門迫須有化解條件說明 | 僅標示問題而無拆解 |
| 用神配對 | 各彩池用神與馬匹落宮關係必須說明 | 漏配 |
| 檔位適性 | DP-036落宮方位與檔位方位比對 | 漏比 |

### 2.2 補丁觸發檢查（奇門相關）

| 補丁 | 觸發條件 | 必須確認 |
|------|----------|----------|
| DP-012 | 所有馬匹 | 十步全覆蓋 |
| DP-016 | 伏吟格+得地 | 是否正確判為不凶 |
| DP-018 | 同宮馬匹 | 是否觸發同宮共享 |
| DP-020 | 所有馬匹 | 十二長生標籤是否僅用於奇門 |
| DP-027v2 | 休門+太陰+內檔 | 是否依宮位調整降級幅度 |
| DP-029 | 全場 | 用神是否鎖定 |
| DP-030 | 所有馬匹 | 雙太極合盤是否執行 |
| DP-031 | 所有馬匹 | 旺衰動靜是否加權 |
| DP-032 | 有四害馬匹 | 能量阻塞是否拆解 |
| DP-033 | 衰+天芮 | 是否觸發雙重警報 |
| DP-036 | 所有馬匹 | 八卦落宮檔位適性是否判定 |
| DP-041v3 | 帝旺馬匹 | 真偽+制化是否判定 |
| DP-042 | 擊刑馬匹 | 起步擠阻是否預警 |
| DP-057v2 | 青龍入海 | 喜忌法則是否判定 |

### 2.3 常見違規

| 違規類型 | 說明 | 處理 |
|----------|------|------|
| 跳步 | 未完成十步即給出結論 | 退回重做 |
| 寄宮遺漏 | 中五宮馬匹未分析寄宮 | 退回重做 |
| 四位割裂 | 僅列舉符號而無一體化解讀 | 退回補充 |
| 帝旺一刀切 | 未區分真旺/虛火/制化 | 退回修正 |


## 三、稽查B：梅花易數SOP合規審查

**執行者**：Fiona  
**審查對象**：階段3.1 梅花易數觀梅占四步SOP  
**審查標準**：依 `sop/sop_meihua.md` 及 `wisdom/meihua.md`

### 3.1 四步覆蓋率審查

| 檢查項 | 標準 | 不合格判定 |
|--------|------|-----------|
| 馬名拆解 | 四字名前二後二分，三字名前二後一 | 拆解錯誤或簡化（違反DP-005/DP-046） |
| 主卦/變卦/動爻 | 必須明確標示 | 漏標 |
| 體用生剋 | 體卦、用卦、生剋關係必須判定 | 漏判或錯判 |
| 時令旺衰 | 月令對體卦的旺衰影響必須判定 | 漏判 |
| 變卦具象化 | 變卦五行必須翻譯為賽馬畫面 | 僅有五行而無翻譯 |
| 三要十應 | 必須有賽馬畫面預測（跑法/障礙/關鍵時刻） | 漏寫 |

### 3.2 補丁觸發檢查（梅花相關）

| 補丁 | 觸發條件 | 必須確認 |
|------|----------|----------|
| DP-005 | 所有馬匹 | 拆解是否正確 |
| DP-013 | 豐卦+≥15倍 | 是否自動復活 |
| DP-017 | 乾卦+低班次 | 是否標註位置最低配置 |
| DP-026 | 剝卦+未贏馬 | 是否降級 |
| DP-034 | 剝卦三爻動 | 是否不受DP-026限制 |
| DP-043 | 益卦 | 是否提升一星 |
| DP-046 | 四字馬名 | 是否全拆 |
| DP-047 | 損→大畜 | 是否判定為領放型 |
| DP-055 | 同人卦 | 是否標註利涉大川 |
| DP-056 | 革卦 | 是否判定為突破 |
| DP-058 | 鼎卦 | 是否標註正位凝命 |
| DP-059v2 | 需→節 | 是否降級+標註待時 |

### 3.3 常見違規

| 違規類型 | 說明 | 處理 |
|----------|------|------|
| 馬名簡化 | 四字馬名只取前兩字起卦 | 退回重做（一級失誤） |
| 體用顛倒 | 動爻所在卦誤判為體卦 | 退回重做 |
| 旺衰遺漏 | 未依月令調整體卦能量 | 退回補充 |
| 補丁遺漏 | 觸發條件滿足但未執行補丁 | 退回修正 |

## 六、階段6：SOP三級審核

**執行者**：Alan（稽查部長）

### 6.1 第一級：執行紀錄完整性

| 檢查項 | 標準 |
|--------|------|
| 階段0-5文件齊全 | 劇本判定書、全局卦象、排盤表、內環分析、稽查證書、外環報告 |
| 格式合規 | 各文件依規定格式填寫 |
| 簽署完整 | 各階段負責人已簽署 |

### 6.2 第二級：內外環邏輯一致性

| 檢查項 | 標準 |
|--------|------|
| 內外環矛盾 | 有無未解決的內外環矛盾 |
| 降維判斷 | 有無被卦象凌駕的降維判斷 |
| 補丁優先級 | 憲法級補丁是否優先執行 |

### 6.3 第三級：終審合規性

| 檢查項 | 標準 |
|--------|------|
| 憲法鐵律遵守 | 七條鐵律無違反 |
| 補丁無衝突 | 觸發的補丁之間無邏輯矛盾 |
| 整體合理性 | 綜合判斷無明顯荒謬之處 |

審查通過後，Alan簽發最終《SOP合規證書》，傳遞至階段7。
