## 二、投票制評級（階段 3.5）

### 2.1 投票規則

每個維度獨立投票，結果轉化為分數：

| 投票結果 | 分數 |
|----------|------|
| 吉（有利） | +1 |
| 平（中性） | 0 |
| 凶（不利） | -1 |

- 騎師卡牌進化後若達「超吉」，則為 **+2**。
- 所有維度權重相等（DP-240）。

### 2.2 總分計算

```

總分 = 梅花 + 奇門陰 + 奇門陽 + 六爻 + 鐵板 + 步速匹配 + 數據部 + 騎師

```

### 2.3 總分 → PTCG 對映表

| 總分範圍 | PTCG | 角色定位 | 投注建議 |
|----------|------|----------|----------|
| +5 至 +8 | UR | 獨贏首選 | 獨贏膽，位置膽 |
| +3 至 +4 | SR | 位置主腳 / 連贏膽 | 位置主腳，連贏配搭 |
| +1 至 +2 | AR | 位置配腳 / 位置Q拖 | 位置配腳，位置Q拖 |
| -1 至 0 | RR | 邊緣配腳 / 四重彩拖 | 僅四重彩拖腳 |
| -3 至 -2 | R | 不建議投注 | 放棄 |
| -4 或以下 | U | 排除 | 強制排除 |

> 若總分 +5 以上但賠率 ≥30 倍，仍需執行冷門復活程序（DP-127）。

## 三、雙環驗證（內環 + 外環）

### 3.1 內環（七術投票）

即上述投票制，判斷馬匹的內在能量與玄學格局。

### 3.2 外環（降維四法）

| 降維方法 | 檢查內容 | 數據來源 |
|----------|----------|----------|
| 場地法 | 檔位、路程、場地適性 | st_rules.md / hv_rules.md |
| 形態法 | 近績、體重、配備、晨操 | 馬匹往績表 |
| 競爭法 | 對手實力、賠率結構 | 賠率表、評分 |
| 人事法 | 騎師能量指數、練馬師部署 | jockey_index.md |

- 外環評級分為 A/B/C/D/E（參見階段 3.8）。
- 外環降維**不可單獨否決內環吉卦**（憲法第二鐵律）。僅當內環凶且外環差時，才強制排除（雙舉證）。

### 3.3 雙重確認交叉表

| 玄學PTCG | 數據評級 | 處理方式 |
|----------|----------|----------|
| UR/SR | A/B | 正向一致 → 可信度倍增，可加重投注 |
| UR/SR | C | 正向部分一致 → 仍信任玄學，標註「數據平庸」 |
| R/U/C | D/E | 負向一致 → 強制排除 |
| UR/SR | D/E | 玄學強數據弱 → 不排除，降一級使用（UR→SR，SR→AR） |
| R/U | A/B | 玄學弱數據強 → 列為「上位殺手」或伏兵候選（SAR） |

## 四、伏兵偵察系統（camp.md 整合）

伏兵分為四級：

| 級別 | 觸發條件 | 評級下限 |
|------|----------|----------|
| A級伏兵 | 鐵板全吉+≥15倍（DP-025）/ 太白入熒坤宮（DP-073） | SAR |
| B級伏兵 | 驚門+玄武（DP-116）/ 杜門+六合短途（DP-062） | SR |
| C級伏兵 | 旬空月破開門（DP-074）/ 同宮冷門（DP-018） | AR |
| D級伏兵 | 能量反轉（DP-105）/ 三T尾關輕磅內檔（DP-106） | RR |

## 五、跨場次能量追蹤（cross_energy.md 整合）

- 追蹤馬匹近三場的**能量趨勢**（向上 ↗↗ / 持平 →→ / 向下 ↘↘ / 反轉 ⇵⇵）。
- 能量反轉（⇵⇵）且騎師 JFI 同步上升 → **伏兵信號 +2**，強制列為 SAR 或以上（DP-105）。

## 六、騎師卡牌進化（jockey_index.md 摘要）

在投票制中，騎師維度的基礎評級由 JFI 決定：

| JFI 範圍 | 基礎評級 | 投票值 |
|----------|----------|--------|
| ≥ 8.5 | S級（頂級） | +1 |
| 6.0 – 8.4 | A/B級（一般） | 0 |
| < 6.0 | C/D級（弱） | -1 |

**進化條件（DP-253）**：滿足任一條件，基礎評級升一級（可疊加，最多升兩級）：

| 編號 | 條件 | 說明 |
|------|------|------|
| E1 | 晨操騎師親試快時間 | 賽前7天內親自快操且時間快於標準0.5秒以上 |
| E2 | 騎師此場地/途程高勝率 | 相同跑道及途程近3年勝率≥15% |
| E3 | 騎師近期狀態火熱 | 近10場勝率≥15% 或 入位率≥40% |
| E4 | 騎師與此馬合作有佳績 | 此前策騎同一馬匹取得前四名 |
| E5 | 騎師擅長變化場地 | 濕慢地/黏地/軟地勝率≥10% |
| E6 | 見習騎師減磅且後上跑法 | 減磅≥7磅 + 馬匹為後上跑法（凶→平） |
| E7 | 騎師此場為「復仇戰」 | 上仗因意外未能發揮 |

進化後投票值：凶→平（0）→吉（+1）→超吉（+2）。

## 七、PTCG 實用性標籤

| 標籤 | 條件 | 投注建議 |
|------|------|----------|
| **主戰力** | UR/SR + 降維無瑕疵 | 獨贏膽 |
| **上位殺手** | RR/SR + 鐵板全吉/同宮大熱+≥15倍 | 冷門獨贏/位置 |
| **實用配搭** | SR/AR + 位置用神吉 | 位置鐵膽 |
| **待開發** | SAR/UR + 降維有瑕疵 | 僅配腳 |
| **小丑** | SAR/SR + 多術矛盾 + 降維差 | 四重彩拖 |
| **凡骨** | C/白板 | 排除 |
