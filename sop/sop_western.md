## 一、總綱

西洋占星為 TJSCODE **階段0劇本輔助判定**流派，採用回歸黃道系統，以心理特質與事件預測並重。其核心任務是從行星、星座、宮位、相位的多重交織中，刻畫賽事當日的整體能量基調，為紫微斗數及印度占星的劇本判定提供第三驗證維度。

**最高心訣**：
> 日為意志月為情，金為悅樂火為爭。  
> 木為擴張土為限，天海冥中藏變驚。  
> 上升定基十宮名，相位交織見輸贏。  
> 互溶接納為貴助，逆行灼傷須警惕。

**與印度占星的分工**：
- 西洋占星：回歸黃道，注重心理動機與事件觸發。
- 印度占星：恆星黃道，注重宿命業力與大運時序。
- 兩者結論可互相印證，矛盾時觸發劇本警報。


## 二、排盤操作流程

### 2.1 排盤參數

| 參數 | 說明 |
|------|------|
| 日期 | 賽事日期（YYYY-MM-DD） |
| 時間 | 開賽時間（香港時間） |
| 地點 | 賽場經緯度（沙田：北緯22°23'，東經114°11'） |
| 黃道系統 | 回歸黃道（Tropical Zodiac） |
| 宮位制 | Placidus 宮位制（或其他常用宮位制） |

### 2.2 排盤工具

可使用 Solar Fire、Astro.com、AstroGold 或其他支援回歸黃道的西洋占星軟體。

### 2.3 排盤步驟

1. 輸入賽事日期、時間、地點。
2. 選擇回歸黃道及 Placidus 宮位制。
3. 生成星盤，記錄以下數據：
   - 上升星座（ASC）及命主星
   - 各行星所在星座及宮位
   - 中天（MC）星座及其主星
   - 主要行星相位（合、六分、四分、三分、對分）
   - 逆行行星
   - 月亮空亡（Void of Course）狀態


## 三、劇本判定六大審查項目

### 3.1 第一審：上升星座（ASC）及命主星

| 審查項 | 內容 |
|--------|------|
| 上升星座 | 決定賽事整體面貌與當日基調 |
| 命主星狀態 | 廟旺/曜升=賽事順利；陷落/失勢=充滿變數 |
| 命主星落宮 | 影響賽事焦點所在 |

**上升星座速查**：

| 上升星座 | 元素 | 模式 | 賽事基調 |
|----------|------|------|----------|
| 白羊 | 火 | 開創 | 競爭激烈，短途爆發 |
| 金牛 | 土 | 固定 | 穩定均速，耐力優先 |
| 雙子 | 風 | 變動 | 靈活多變，走位飄忽 |
| 巨蟹 | 水 | 開創 | 主場優勢，熟悉場地 |
| 獅子 | 火 | 固定 | 明星焦點，大熱可信 |
| 處女 | 土 | 變動 | 技術分析，戰術執行 |
| 天秤 | 風 | 開創 | 合作協調，騎練配合 |
| 天蠍 | 水 | 固定 | 伏兵暗藏，殺手本能 |
| 射手 | 火 | 變動 | 長途探索，外來馬 |
| 摩羯 | 土 | 開創 | 紀律策略，經驗老馬 |
| 水瓶 | 風 | 固定 | 創新冷門，非傳統 |
| 雙魚 | 水 | 變動 | 混亂不確定，靈感 |

### 3.2 第二審：第十宮（MC）及事業宮主星

| 審查項 | 內容 |
|--------|------|
| MC星座 | 決定「勝利者」的公眾形象 |
| 第十宮主星 | 揭示勝利者的特質 |
| 第十宮內行星 | 影響勝利者的能量 |

**MC主星速查**：

| MC主星 | 勝利者特質 |
|--------|-----------|
| 火星 | 短途、爆發、前領型 |
| 金星 | 流暢、悅目、明星馬 |
| 木星 | 長途、幸運、貴人相助 |
| 土星 | 經驗、紀律、老馬 |
| 太陽 | 權威、王者、級際賽 |
| 月亮 | 公眾情緒、主場優勢 |
| 水星 | 技術、靈活、彎位專家 |

### 3.3 第三審：月亮狀態

| 審查項 | 內容 |
|--------|------|
| 月亮星座 | 當日情緒與公眾氛圍 |
| 月亮相位 | 與土星/天王/海王的硬相位=冷門或混亂 |
| 月亮空亡 | 賽事易出意料結果，冷門信號 |

### 3.4 第四審：冷門信號（第五宮及第八宮）

| 審查項 | 內容 |
|--------|------|
| 第五宮主星 | 與天王星有相位=冷門投機 |
| 第五宮內行星 | 天王星在第五宮=突發冷門 |
| 第八宮主星 | 與冥王星/火星有相位=意外事件 |
| 第八宮內行星 | 冥王星在第八宮=極端賽果 |

### 3.5 第五審：木星與金星（吉星）

| 審查項 | 內容 |
|--------|------|
| 木星位置 | 守護財帛，代表幸運所在 |
| 金星位置 | 流暢與協調，代表狀態上佳 |
| 木金相位 | 合相/三分=大吉，派彩豐厚 |

### 3.6 第六審：逆行與灼傷

| 審查項 | 內容 |
|--------|------|
| 水星逆行 | 騎師指令失誤、馬匹反應遲緩 |
| 火星逆行 | 競爭能量內斂，可能反常跑法 |
| 行星灼傷 | 能量被太陽遮蔽，難以發揮 |


## 四、冷門指數計算

### 4.1 加分項（冷門信號）

| 信號 | 加分 |
|------|------|
| 月亮空亡 | +3 |
| 天王星在第五宮/第八宮 | +2 |
| 月亮與天王星/海王星硬相位 | +2 |
| 上升為水象固定宮（天蠍） | +1 |
| 水星逆行 | +1 |
| 命主星陷落/失勢 | +1 |

### 4.2 減分項（正路信號）

| 信號 | 減分 |
|------|------|
| 木星與金星合相/三分相 | -2 |
| 上升為火象固定宮（獅子） | -1 |
| 命主星廟旺 | -1 |
| MC主星廟旺 | -1 |

### 4.3 冷門指數對照

| 指數 | 級別 | 處理 |
|------|------|------|
| 8-10 | 🔴 極高 | 冷門活躍，伏兵必出 |
| 5-7 | 🟡 偏高 | 冷門有機會 |
| 2-4 | 🟢 中等 | 正路為主 |
| 0-1 | 🔵 偏低 | 正路賽果，大熱可信 |
