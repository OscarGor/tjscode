## 一、場地概覽

跑馬地（Happy Valley）為香港島市中心的草地賽道，賽道呈 **順時針方向**，全長約 **1450 米**，彎急路窄，極具戰術性。

| 特性 | 說明 |
|------|------|
| 方向 | 順時針（右轉） |
| 賽道總長 | 約 1450 米 |
| 直路長度 | 約 300 米 |
| 彎位數量 | 兩個主要彎角（第一彎、第二彎） |
| 路面闊度 | 約 20-25 米（較沙田窄） |
| 場地類型 | 天然草地 |
| 排水系統 | 良好，但連場大雨後可出現軟爛地 |

## 二、檔位偏差特性

跑馬地因彎急路窄，**檔位對賽果的影響遠大於沙田**。內檔馬匹在彎位可節省路程，外檔馬匹則需繞大圈，耗力較多。

### 2.1 各路程檔位優劣

| 路程 | 最佳檔位 | 不利檔位 | 說明 |
|------|----------|----------|------|
| 1000 米 | 1-4 檔 | 9-12 檔 | 起步即轉彎，外檔幾乎必大外避，形同自殺 |
| 1200 米 | 1-6 檔 | 10-12 檔 | 首彎前短直路，外檔馬需搶前或留後，兩難局面 |
| 1650 米 | 1-5 檔 | 10-12 檔 | 中距離，內檔可守好位，外檔需消耗更多體力 |
| 1800 米 | 3-8 檔 | 1-2 檔、11-12 檔 | 極內檔可能被困，極外檔耗力大 |
| 2200 米 | 2-7 檔 | 1 檔、11-12 檔 | 長途，檔位影響稍減，但仍以中檔為佳 |

### 2.2 檔位玄學對應（與奇門八門關聯）

依 `sop/sop_qimen.md`，跑馬谷彎急，**杜門、休門** 落宮之馬在此場地最具優勢：

| 八門 | 對應檔位傾向 | 原因 |
|------|-------------|------|
| 杜門 | 中內檔（3-7 檔） | 隱藏實力，轉彎時可沿欄省位 |
| 休門 | 內檔（1-4 檔） | 養精蓄銳，留前鬥後 |
| 生門 | 中檔（4-8 檔） | 生機在轉彎後發力 |
| 開門 | 中檔（5-9 檔） | 暢通但需繞外，只利前領型 |
| 驚門 | 外檔（9-12 檔） | 心神不寧，轉彎易出錯 |
| 死門 | 極外檔（10-12 檔） | 前路不通，多被阻塞 |

## 三、跑道分段特性

### 3.1 起步段（0-300 米）

- 1000 米賽事：起步後 **不足 200 米即進入首彎**，外檔馬極度不利。
- 1200 米賽事：起步後有約 400 米直路才入彎，外檔馬有時間搶前或留後。
- 1650 米或以上：起步段相對從容，檔位影響下降。

### 3.2 首彎（第一彎角）

- 彎度約 **120 度**，路面收窄。
- 外檔馬在此彎 **被迫大外避**，每外移一檔約損失 1-1.5 馬位。
- 內檔馬可沿欄慳位，優勢明顯。

### 3.3 對面直路（600-900 米段）

- 相對平坦，馬匹可在此段調整呼吸及位置。
- 此段 **步速變化** 是決勝關鍵：步速過慢利前領馬，步速過快利後追馬。

### 3.4 第二彎及入直路（900-1200 米段）

- 第二彎轉入最後直路，彎度較首彎緩和。
- 外檔馬若在此彎前未能取得遮擋，將被迫提早發力，末段容易無以為繼。

### 3.5 最後直路（最後 300 米）

- 直路長約 300 米，較沙田短 150 米。
- **後追馬需在轉彎前已開始加速**，否則難以在短直路追上。
- 領放馬若能在轉彎時帶出 2-3 個馬位，勝出機會極高。

## 四、場地狀況影響

### 4.1 好地（Good）

標準狀況，以上檔位及跑道特性全面適用。

### 4.2 好至快地（Good to Firm）

- 場地偏快，**前領馬優勢增加**。
- 內檔馬慳位優勢更明顯。
- 短途賽（1000/1200）領放馬極難被追上。

### 4.3 好至黏地（Good to Yielding）/ 軟地（Soft）

- 場地有水分，**後追馬較易發揮**。
- 外檔馬劣勢稍減，因場地較軟可吸收部分外避的動能損失。
- 泥地血統馬或曾在軟地勝出的馬匹優先考慮。

### 4.4 大雨 / 軟爛地（Heavy）

- 場地極度考驗耐力，**均速長途馬有利**。
- 領放馬在軟爛地易體力透支。
- 內檔場地往往較外檔更爛（因水往低流），極內檔（1-2 檔）反而不利。
- **奇門坎宮（水）及休門落宮之馬**在此狀況下加分。

## 五、玄學場地關聯

### 5.1 梅花易數與快活谷

- **萃卦䷬**在快活谷有特殊效應，詳見補丁 DP-007。
- 體卦為 **震木**（爆發）之馬，在谷草短途（1000/1200）有優勢。
- 體卦為 **坤土**（耐力）之馬，在谷草長途（1800/2200）有優勢。

### 5.2 奇門遁甲與快活谷

快活谷賽事中，以下奇門組合特別有利：

| 組合 | 含義 |
|------|------|
| 杜門 + 太陰 | 暗藏殺機，伏兵之兆，冷門溫床 |
| 休門 + 六合 | 留前鬥後，配合完美，連贏佳選 |
| 驚門 + 白虎 | 心神不寧，易出意外，熱門高危 |

### 5.3 九星氣學與快活谷

- 日盤九星為 **四綠木星** 時，技術型騎師在谷草有優勢。
- 騎師本命星與日盤構成 **木木比和**（如三碧 + 四綠），利谷草多彎場合。

## 六、部署建議摘要

| 賽事類型 | 核心策略 |
|----------|----------|
| 谷草 1000 米 | 優先內檔前領馬，外檔馬只可作伏兵考慮 |
| 谷草 1200 米 | 中內檔可攻可守，留意步速變化 |
| 谷草 1650 米 | 內檔好位馬為膽，外檔後追馬為配腳 |
| 谷草 1800/2200 米 | 檔位影響下降，以體卦耐力及騎師氣場為重 |
| 谷草軟地 | 後追馬加分，內檔反而不利 |
| 谷草夜賽 | 光線、氣溫影響，九星氣學的「月亮宿度」特別重要 |
