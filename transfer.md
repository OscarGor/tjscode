## 二、憲法級鐵律（不可違反）

1. **體用為王**：梅花體用生剋是所有卦象解讀的最高原則。當梅花卦象出現結構性缺陷（觀→剝、歸妹→睽），其他流派的共振信號不可掩蓋梅花的警告。**卦德可凌駕體用**（姤、同人、家人、革、鼎、震、師、豐、渙、節等）。
2. **卦象直指凌駕一切**：卦象明確信號不可被數據對沖或降權。
3. **$200 紀律**：每場總投注固定為港幣 $200，不可加減。
4. **SOP 強制執行**：所有流程不可跳步，不可自作聰明。
5. **排除須雙舉證**：排除任何馬匹必須同時提供內環及外環舉證。
6. **冷門強制復活**：30倍以上冷門馬強制反向質疑復活；≥15倍且特定卦象自動復活；冷門強度≥6分時復活門檻降至15倍；**三T尾關降至10倍**；**≥20倍＋驚門玄武強制復活入圍（DP-257）**。
7. **七術全覆蓋**：每場賽事必須完成全部七術分析，缺一不可。
8. **雙重確認**：玄學信號與數據信號須進行交叉驗證。正向一致時可信度倍增；負向一致時排除決策更堅定。
9. **投票制統籌**：最終 PTCG 由多術數投票總分決定（DP-239）。投票維度包括：梅花、奇門陰陽、六爻、鐵板、步速匹配、數據部、騎師（含進化）。
10. **步速匹配不可跳過**：階段2.5必須執行步速預測，否則視為 SOP 違規（DP-232）。
11. **奇門用神優先（Ver.2.8 新增）**：奇門用神格局（景門值符、驚門玄武、生門六合、開門九天）為「天時」信號，優於紫微斗數命宮之「命格」判斷。當兩者衝突時，**以奇門用神為準**（DP-256）。
12. **大熱過熱警告（Ver.2.8 新增）**：賠率 ≤ 3倍 且出現白虎剋身／體重低於平均-40磅／卦象「既濟䷾初吉終亂」，自動降1級評級（DP-258）。

## 二、強制執行流程（九階段）

### 階段 0：劇本五層穿透（含前置過濾器）
**執行者**：卦象部  
**目標**：鎖定賽事天時能量，判定正路/冷門傾向、冷門強度、頭馬類型及檔位分佈。

#### 0.0 前置過濾器：班次權重與實力壓制
- **一級賽冷門強度上限**：G1 賽事強制鎖定冷門強度 ≤5/10。
- **實力壓制**：馬匹評分 ≥ 全場平均 +15 且近三仗全入前二 → 劇本強制正路，冷門強度減半。

#### 0.1 紫微斗數排盤
- 依賽事時間起紫微命盤，分析命宮、子女、財帛、官祿、遷移、父母等宮位。
- 輸出《紫微劇本判定書》。

#### 0.2 印度占星排盤
- 以恆星黃道起盤，計算月亮宿度、羅睺/計都位置、行星相位。
- 輸出《印度占星劇本判定書》。

#### 0.3 劇本五層穿透綜合
| 層次 | 內容 |
|------|------|
| 第一層 | 正路/冷門傾向（已受前置過濾器修正） |
| 第二層 | 冷門強度（0-10分），三T尾關≥4分啟動半熱門審查 |
| 第三層 | 冷門類型（真空奪位/暗中伏兵/破舊立新/困局突破） |
| 第四層 | 頭馬類型（實力大熱/伏兵殺手/純粹伏兵） |
| 第五層 | 1-4名檔位分佈（依場地及步速動態調整） |

- 紫微與印度結論矛盾時，觸發劇本警報，冷門復活門檻降至15倍。
- 輸出《綜合劇本判定書》。

#### 0.4 奇門用神格局檢查（Ver.2.8 新增）

| 驚門+玄武（≥20倍） | 🥈 次高 | 強制復活入圍 | 自動 +3 |
| 開門+九天 | 🥉 高 | 穩定入位 | 自動 +2 |
| 生門+六合 | 中高 | 伏兵入位 | 自動 +2 |

**執行規則**：若奇門格局與紫微+印度結論衝突，**以奇門用神為準**（憲法第十一鐵律）。記錄偏差原因，納入階段7旁觀者視角仲裁。

### 階段 2.5：步速預測與跑法匹配
**執行者**：卦象部  
**強制執行**（DP-232），不可跳步。

#### 2.5.1 步速預測方法
| 方法 | 操作 | 輸出 |
|------|------|------|
| 時間法 | 參考前800米標準分段（如1600米約47.5秒）。若有多匹領放馬，傾向快步速 | 快步速/慢步速/標準 |
| 奇門法 | 值使門為驚門/死門且落兌宮 → 快步速；值使門為休門/生門且落坎/艮 → 慢步速 | 同上 |
| 梅花法 | 師卦、震卦 → 快步速；豫卦、漸卦 → 慢步速；觀卦 → 標準 | 同上 |
| 六爻法（動爻） | 動爻數≥3 → 快步速；≤1 → 慢步速；2 → 標準（DP-235） | 同上 |

- 若多術數矛盾，以奇門權重最高（DP-237）。

#### 2.5.2 步速主宰者雙重驗證（Ver.2.8 新增）
必須同時確認：
1. 邊匹馬會放頭（近績走位＋騎師風格）
2. 邊匹馬會跟跑（近績走位＋檔位）

驗證方法：對比「預測放頭馬」與「近績走位1-1-1/1-1-2嘅馬」是否一致。若不一致，重新評估步速預測。

#### 2.5.3 跑法匹配調整
| 步速類型 | 有利跑法 | 不利跑法 |
|----------|----------|----------|
| 快步速 | 後上、中置 | 領放、前置 |
| 慢步速 | 領放、前置 | 後上、中置 |
| 標準 | 均衡 | 無 |

- **調整規則**：有利跑法的馬匹在投票制中「步速匹配」維度得 **+1**（吉），不利跑法得 **-1**（凶）。DP-233、DP-234。

#### 2.5.4 輸出
- 步速預測結論（含依據）。
- 每匹馬的「步速匹配」投票結果（+1/0/-1）。

- DP-030 雙太極合盤（馬匹落宮＋騎師定宮）。
- 輸出鐵板投票（吉/平/凶）。

#### 3.7 騎師卡牌進化（動態調整騎師維度）
- 基礎騎師 JFI 對應：≥8.5→吉（+1），6.0-8.4→平（0），<6.0→凶（-1）。
- 觸發進化條件（DP-253）：
  - E1：晨操騎師親試快時間 → 升一級
  - E2：騎師此場地/途程高勝率（≥15%）→ 升一級
  - E3：騎師近10場勝率≥15%或入位率≥40% → 升一級
  - E4：騎師與此馬合作有前四名記錄 → 升一級
  - E5：騎師擅長變化場地 → 升一級
  - E6：見習騎師減磅≥7磅且後上跑法 → 凶→平
  - E7：騎師此場為「復仇戰」→ 升一級
- 最終騎師投票值為進化後對應值（凶→平→吉→超吉+2）。


### 階段 3.5：多術數投票制評級（強制）

## 二、投票制評級（階段 3.5）

### 2.1 投票規則

每個維度獨立投票，結果轉化為分數：

| 投票結果 | 分數 |
|----------|------|
| 吉（有利） | +1 |
| 平（中性） | 0 |
| 凶（不利） | -1 |

- 騎師卡牌進化後若達「超吉」，則為 **+2**。
- 所有維度權重相等（DP-240）。

### 2.2 奇門用神優先級（Ver.2.8 新增 — 憲法第十一鐵律）

奇門用神格局為「天時」信號，優於紫微斗數命宮之「命格」判斷。**當兩者衝突時，以奇門用神為準**（DP-256）。

| 奇門格局 | 優先級 | 角色定位 | 投票修正 |
|----------|--------|----------|----------|
| 景門+值符 | 🥇 最高 | 冠軍首選 | 自動 +3 |
| 驚門+玄武（賠率≥20倍） | 🥈 次高 | 強制復活入圍 | 自動 +3 |
| 開門+九天 | 🥉 高 | 穩定入位 | 自動 +2 |
| 生門+六合 | 中高 | 伏兵入位 | 自動 +2 |

### 2.3 大熱過熱檢查（Ver.2.8 新增 — 憲法第十二鐵律）

**觸發條件**：賠率 ≤ 3倍 且 出現以下任一信號：
- 奇門白虎剋身（坎宮休門+白虎）
- 體重低於同場平均-40磅
- 卦象「既濟䷾初吉終亂」

**執行結果**（DP-258）：
- 自動降1級評級（UR→SR，SR→AR）
- 在評級表中標註「⚠️ 大熱過熱警告」
- 不得作為獨贏首選

## 六、騎師卡牌進化（jockey_index.md 摘要）

在投票制中，騎師維度的基礎評級由 JFI 決定：

| JFI 範圍 | 基礎評級 | 投票值 |
|----------|----------|--------|
| ≥ 8.5 | S級（頂級） | +1 |
| 6.0 – 8.4 | A/B級（一般） | 0 |
| < 6.0 | C/D級（弱） | -1 |

**進化條件（DP-253）**：滿足任一條件，基礎評級升一級（可疊加，最多升兩級）：

| 編號 | 條件 | 說明 |
|------|------|------|
| E1 | 晨操騎師親試快時間 | 賽前7天內親自快操且時間快於標準0.5秒以上 |
| E2 | 騎師此場地/途程高勝率 | 相同跑道及途程近3年勝率≥15% |
| E3 | 騎師近期狀態火熱 | 近10場勝率≥15% 或 入位率≥40% |
| E4 | 騎師與此馬合作有佳績 | 此前策騎同一馬匹取得前四名 |
| E5 | 騎師擅長變化場地 | 濕慢地/黏地/軟地勝率≥10% |
| E6 | 見習騎師減磅且後上跑法 | 減磅≥7磅 + 馬匹為後上跑法（凶→平） |
| E7 | 騎師此場為「復仇戰」 | 上仗因意外未能發揮 |

進化後投票值：凶→平（0）→吉（+1）→超吉（+2）。

## Ver.2.8 新增補丁（DP-256 至 DP-259）

#### DP-256：奇門用神優先級制度（憲法級｜Ver.2.8 新增）

**觸發條件**：任何賽事中，奇門用神格局（景門值符、驚門玄武、生門六合、開門九天）與紫微斗數命宮判斷衝突時。

**執行內容**：
- 以奇門用神為準，紫微命宮僅作參考。
- 景門+值符 → 自動 +3，列為冠軍首選。
- 驚門+玄武（賠率≥20倍）→ 自動 +3，強制復活入圍，評級不得低於 SR。
- 開門+九天 → 自動 +2，列為穩定入位。
- 生門+六合 → 自動 +2，列為伏兵入位。

**違規後果**：一級操作失誤

**對應鐵律**：憲法第十一鐵律

**判例**：2026-06-07 第10場 8號喜慶寶（景門值符，原紫微命宮判斷為回落期，奇門優先後修正為冠軍）


#### DP-257：驚門玄武冷門強制復活（憲法級｜Ver.2.8 新增）

**觸發條件**：奇門格局 = 驚門+玄武 且 賠率 ≥ 20倍。

**執行內容**：
- 強制復活入圍，無須執行逆向質疑五問程序。
- 復活後評級不得低於 SR。
- 在投票制中給予 +3 修正。

**違規後果**：一級操作失誤

**對應鐵律**：憲法第六鐵律、第十一鐵律

**判例**：2026-06-07 第10場 5號魔術控制（驚門玄武+27倍，原被排除，修正後列為亞軍）


#### DP-258：大熱過熱警告（憲法級｜Ver.2.8 新增）

**觸發條件**：賠率 ≤ 3倍 且 出現以下任一信號：
- 奇門白虎剋身（坎宮休門+白虎）
- 體重低於同場平均-40磅
- 卦象「既濟䷾初吉終亂」

**執行內容**：
- 自動降1級評級（UR→SR，SR→AR）
- 在預測報告中標註「⚠️ 大熱過熱警告」
- 不得作為獨贏首選

**違規後果**：二級操作失誤

**對應鐵律**：憲法第十二鐵律

**判例**：2026-06-07 第10場 6號扶搖勢勁（2.6倍，體重1104磅低於平均-40磅，原列冠軍，修正後降級）


#### DP-259：冷門＋奇門格局共振（憲法級｜Ver.2.8 新增）

**觸發條件**：賠率 ≥ 20倍 且 奇門格局為驚門玄武或生門六合。

**執行內容**：
- 強制納入頭四名考慮，不可排除。
- 在投票制中給予 +2 修正。
- 若同時符合 DP-257 條件，疊加生效（共 +5）。

**違規後果**：一級操作失誤

**對應鐵律**：憲法第六鐵律、第十一鐵律

**判例**：2026-06-07 第10場 5號魔術控制（驚門玄武+27倍，修正後列為亞軍）
