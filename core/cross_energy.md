## 一、心法總綱

跨場次能量追蹤為 TJSCODE **外環動態監測**的核心模組。如果說 `core/aura.md` 是馬匹當下狀態的「定格照片」，本系統則是馬匹、騎師、馬房近期走勢的「連續影片」。它專注於捕捉能量的**趨勢轉折**——尤其是在低谷中突然抬頭的「伏兵信號」。

**最高心訣**：
> 孤場不判，三場成勢。  
> 能量反轉，伏兵將至。  
> 騎師連勝，氣貫長虹。  
> 馬房集結，聯攻可期。  
> 三T尾關，反轉加倍。  
> 輕磅內檔，冷門必追。

## 二、追蹤維度與指標

### 2.1 三維追蹤

| 追蹤對象 | 核心指標 | 數據來源 |
|----------|----------|----------|
| **馬匹** | 近3-6場 Aura 能量指數變化 | `core/aura.md` 歷史記錄 |
| **騎師** | 近5場勝率/入圍率 + 九星氣場走勢 + JFI | `data/jockey_index.md`、`wisdom/jiuxing.md` |
| **馬房** | 近5場整體表現 + 同場多匹馬配合 + SFI | 賽果數據庫 |

### 2.2 能量趨勢定義

| 趨勢符號 | 定義 | 賽馬含義 |
|----------|------|----------|
| ↗↗ | 連續兩場或以上 Aura 指數上升 | 強勢上升期 |
| ↗ | 輕微上升 | 緩步向好 |
| →→ | 指數持平（±0.5分內） | 穩定 |
| ↘ | 輕微下滑 | 狀態回落 |
| ↘↘ | 連續兩場或以上下滑 | 衰退期 |
| ⇵⇵ | 長期下滑或低迷後突然回升 | **反轉信號**（伏兵重點） |

## 三、馬匹能量追蹤

### 3.1 走勢分析

- 取近3場 Aura 指數（AI），繪製走勢圖。
- 判定當前能量趨勢符號。
- 若走勢與內環卦象出現「背離」，須特別警惕（如卦象極強但能量走下坡，可能是「虛火」）。

### 3.2 反轉信號 ⇵⇵ 的確認

| 條件 | 說明 |
|------|------|
| 前3場 AI 呈下降或低迷（均 < 5.0） | 前期低谷 |
| 本場 AI 回升至 6.0 以上 | 明顯回升 |
| 卦象出現復甦信號（剝→晉、明夷→復、困→解、需→節、師→蒙、震為雷） | 卦象佐證 |
| 十二長生為「胎」或「養」 | 奇門蓄力佐證 |
| 負磅 ≤118 且檔位 1-4（三T尾關適用） | 現實條件加持（DP-106） |

**確認 ⇵⇵ 後**：
- 該馬自動進入伏兵兵營（`core/camp.md`），伏兵信號 **+2**（原+1，因實戰驗證權重提升）。
- 若為三T尾關，伏兵級別再提升一級（DP-104）。

## 四、騎師能量追蹤

### 4.1 騎師狀態指數 (Jockey Form Index, JFI)

| 指標 | 權重 | 說明 |
|------|------|------|
| 近5場勝率 | 30% | 近期勝出能力 |
| 近5場入圍率（前三） | 25% | 穩定入圍能力 |
| 九星氣場當日級別 | 25% | 天時能量加持 |
| 與特定馬匹合作往績 | 20% | 人馬默契度 |

**JFI 計算示例**（2026-05-24 第7場前）：
- 潘頓：近5場勝率25%，入圍率55%，九星A級，合作往績佳 → JFI 原高，但當日走勢 ↘↘ → 最終座騎大敗
- 巫顯東：近5場勝率5%，入圍率15%，九星C級，合作一般 → JFI 低，但走勢 ↗ → 座騎爆冷

### 4.2 騎師趨勢判定

| JFI 走勢 | 符號 | 處理 |
|----------|------|------|
| JFI 連續上升 | ↗↗ | 該騎師所有座騎加分，伏兵信號+1 |
| JFI 連續下滑 | ↘↘ | 該騎師所有座騎降級（尤其大熱門，DP-099亢龍有悔） |
| JFI 與馬匹 AI 同升 | 同步 | 人馬能量共振，加半星，伏兵級別升一級 |
| JFI 與馬匹 AI 背離 | 背離 | 可能人馬不合，減半星，排除或降級 |

### 4.3 判例

- **潘頓（第7場）**：JFI ↘↘ + 座騎2號友瑩光（大熱） → 大敗14名
- **巫顯東（第7場）**：JFI ↗ + 座騎12號勁大威猛（17倍） → 冠軍
- **蔡明紹（第7場）**：JFI →→ + 座騎13號勁好運（61倍） → 亞軍

## 五、馬房能量追蹤

### 5.1 馬房集結信號

- 同馬房有 2 匹或以上馬匹出賽同一場次。
- 若其中一匹的卦象/能量信號極強，同馬房其他馬匹可能擔當「掩護」角色。
- 奇門六合同宮或與馬房主帥座騎同宮者，可提升為配腳候選。

### 5.2 馬房狀態指數 (Stable Form Index, SFI)

| 指標 | 說明 |
|------|------|
| 近5場勝率 | 馬房整體贏馬能力 |
| 近5場入圍率 | 馬房穩定度 |
| 本場出馬數 | 兵力集中度 |
| 跨場次趨勢 | 馬房整體能量走向 |

### 5.3 判例

- **廖康銘（第7場）**：出馬3匹，大熱2號大敗，冷門13號亞軍 → 馬房內部「掩護效應」
- **姚本輝（第7場）**：僅出1匹12號，SFI ↗，馬房反彈 → 冠軍
- **方嘉柏（第6場）**：出馬2匹，5號冠軍、3號大敗（不同場次）

## 六、跨場次能量與伏兵偵察聯動

### 6.1 聯動規則（Ver.2.3 強化）

| 跨場次信號 | 伏兵系統對應 |
|------------|-------------|
| 馬匹 ⇵⇵ 反轉 | 伏兵信號 **+2**（直接入B級或以上） |
| 三T尾關之 ⇵⇵ 反轉 | 伏兵信號 **+3**，強制納入位置考慮（DP-104） |
| 騎師 ↗↗ 強勢 | 該騎師所有 ≥15倍座騎伏兵信號 +1 |
| 馬房集結 + 馬匹 AI 上升 | 伏兵信號 +1 |
| 能量與卦象背離 | 伏兵信號 -1（可能虛火） |
| 人馬共振（JFI + AI 同步上升） | 伏兵信號 +2，可做獨贏膽 |

### 6.2 判例

- **仁仁有餘**：⇵⇵反轉 + 莫雷拉JFI↗↗ + 卦象師→蒙 → 伏兵信號累積+4 → A級伏兵 → 冠軍
- **勁好運**：⇵⇵反轉 + 蔡明紹JFI→→ + 卦象同人→家人 + 三T尾關 → 伏兵信號+5 → A級伏兵 → 亞軍
- **勁大威猛**：⇵⇵反轉 + 巫顯東JFI↗ + 震卦 + 輕磅內檔 + 三T尾關 → 伏兵信號+4 → A級伏兵 → 冠軍
