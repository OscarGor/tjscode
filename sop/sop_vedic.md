## 一、總綱

印度占星（吠陀占星）為 TJSCODE **階段0劇本雙重判定**的核心流派之一。其任務是在賽事前，從恆星黃道天象層面鎖定當日的整體能量基調，判定冷門信號強度、有利跑法、路程及檔位傾向。

**最高心訣**：
> 九曜為演員，十二宮為舞台。  
> 羅喉主突變，計都藏因果。  
> 大吉在木金，大凶在火土。  
> 月宿定性情，相位見成敗。

**與西洋占星的核心差異**：
- 使用恆星黃道（Sidereal Zodiac），因歲差與西洋回歸黃道差約24度。
- 月亮宿度（Nakshatra）系統比十二星座更精細。
- 大運系統（Dashas）可推算狀態時間節點。
- 分盤系統（Vargas）可進行多維驗證。


## 二、排盤操作流程

### 2.1 排盤參數

| 參數 | 說明 |
|------|------|
| 日期 | 賽事日期（YYYY-MM-DD） |
| 時間 | 開賽時間（香港時間，轉GMT+8） |
| 地點 | 賽場經緯度（沙田：北緯22°23'，東經114°11'） |
| 宮位制 | 整宮制（Whole Sign House System） |
| 黃道系統 | 恆星黃道（Sidereal Zodiac，Lahiri Ayanamsa） |

### 2.2 排盤工具

可使用 Jagannatha Hora、AstroSage、或其他支援恆星黃道的吠陀占星軟體。

### 2.3 排盤步驟

1. 輸入賽事日期、時間、地點。
2. 選擇恆星黃道及整宮制。
3. 生成星盤，記錄以下數據：
   - 上升星座（Lagna / Ascendant）
   - 各行星所在星座及宮位
   - 月亮所在宿度（Nakshatra）
   - 主要行星相位（Drishti）


## 三、劇本判定六大審查項目

### 3.1 第一審：命宮（Lagna）審查

| 審查項 | 內容 |
|--------|------|
| 命宮星座 | 決定賽事整體氣質 |
| 命宮內行星 | 吉星入命（木、金）→ 賽事正路；凶星入命（火、土、羅、計）→ 易出冷門 |
| 命主星力量 | 命主星強 → 賽事順利；命主星弱 → 賽事多變數 |

### 3.2 第二審：月亮宿度（Nakshatra）審查

| 審查項 | 內容 |
|--------|------|
| 當日月宿 | 決定賽場整體氣質 |
| 星宿類型 | 尖銳型（競爭力強）、穩定型（正路）、波動型（冷門）、隱伏型（伏兵） |

**星宿類型速查**：

| 類型 | 代表宿 | 賽馬應用 |
|------|--------|----------|
| 尖銳型 | Jyeshtha, Mula, Ardra | 競爭力強，利實力派 |
| 穩定型 | Rohini, Pushya, Uttara Phalguni | 正路為主，大熱可信 |
| 波動型 | Ardra, Swati, Shatabhisha | 狀態飄忽，冷門溫床 |
| 隱伏型 | Ashlesha, Mula, Revati | 暗藏實力，伏兵活躍 |

### 3.3 第三審：冷門信號審查（第五宮及第八宮）

| 審查項 | 內容 |
|--------|------|
| 羅喉位置 | 在第五宮（投機宮）→ 冷門信號強，爆冷指數+2 |
| 水星位置 | 在第八宮（意外宮）→ 突發事件，爆冷指數+1 |
| 第五宮主星 | 與凶星（火/土/羅/計）有相位 → 冷門信號 |
| 第八宮主星 | 與天王星/海王星/冥王星有相位 → 突發事件 |

### 3.4 第四審：財帛與收益審查（第二宮及第十一宮）

| 審查項 | 內容 |
|--------|------|
| 木星位置 | 在第二宮（財帛宮）→ 財帛守護，整體收益可期 |
| 火星位置 | 在第十一宮（收益宮）→ 收益有動力但非一帆風順 |
| 金星位置 | 與木星同宮或互溶 → 大吉之合，派彩豐厚 |

### 3.5 第五審：阻力與障礙審查（第六宮）

| 審查項 | 內容 |
|--------|------|
| 土星位置 | 在第六宮（競爭宮）→ 競爭中有暗中阻力 |
| 土星相位 | 對沖月亮 → 熱門馬情緒壓力，狀態不穩 |
| 第六宮主星 | 與凶星合相 → 障礙增加 |

### 3.6 第六審：相位（Drishti）審查

| 審查項 | 內容 |
|--------|------|
| 木星相位 | 對哪些宮位投射相位（5、7、9宮）→ 幸運降臨何處 |
| 土星相位 | 對哪些宮位投射相位（3、7、10宮）→ 阻力降臨何處 |
| 火星相位 | 對哪些宮位投射相位（4、7、8宮）→ 競爭能量何處 |


## 四、爆冷指數計算

### 4.1 計算公式

爆冷指數 = 基礎值(5) + 冷門信號加分 - 正路信號減分

### 4.2 加分項（冷門信號）

| 信號 | 加分 |
|------|------|
| 羅喉在第五宮 | +2 |
| 水星在第八宮 | +1 |
| 月亮在波動型/隱伏型星宿 | +1 |
| 第五宮主星與凶星有相位 | +1 |
| 命宮內有凶星（火/土/羅/計） | +1 |
| 土星對沖月亮 | +1 |
| **太陽與羅睺合相（緊密≤10°）** | **+2** |
| **太陽與羅睺合相（寬鬆≤20°）** | **+1** |

### 4.3 減分項（正路信號）

| 信號 | 減分 |
|------|------|
| 木星在第二宮 | -1 |
| 木星與金星合相 | -1 |
| 月亮在穩定型星宿 | -1 |
| 命宮內有吉星（木/金） | -1 |

### 4.4 爆冷指數對照

| 指數 | 級別 | 處理 |
|------|------|------|
| 8-10 | 🔴 極高 | 冷門活躍，伏兵必出，冷門復活門檻降至15倍 |
| 6-7 | 🟡 偏高 | 冷門有機會，≥30倍強制復活 |
| 4-5 | 🟢 中等 | 正路為主，冷門為輔 |
| 1-3 | 🔵 偏低 | 正路賽果，大熱可信 |

|----------|----------|----------|

## 六、業力消解應用（DP-038）

### 6.1 觸發條件

馬匹遭遇以下逆境，且卦象顯示復甦信號：

| 逆境 | 復甦卦象 |
|------|----------|
| 內檔不利 | 剝→晉、需→節 |
| 負重增加 | 明夷→復、困→解 |
| 配備變更 | 革卦、蠱卦 |
| 外檔劣勢 | 同人卦、益卦 |

### 6.2 處理程序

- 逆境不解讀為「排除因素」，而解讀為「業力消解過程」。
- 納入位置/四重彩配腳，而非排除。
- 如卦象復甦信號明確，伏兵級別提升一級。


## 七、Ver.2.5 新增：太陽與羅睺合相（日月蝕瑜珈）

### 7.1 定義

- 太陽與羅睺在同一星座，且度數差 ≤10°（**緊密合相**）或 ≤20°（**寬鬆合相**）
- 此配置稱為「Grahan Yoga」或「Shakti Yoga」

### 7.2 賽馬應用映射

| 合相類型 | 冷門強度影響 | 大熱處理 | 備註 |
|----------|--------------|----------|------|
| 緊密合相（≤10°） | 爆冷指數 +2；冷門強度基礎+2 | 大熱（≤3倍）強制降級至RR | 大熱倒灶風險極高 |
| 寬鬆合相（11°-20°） | 爆冷指數 +1；冷門強度基礎+1 | 大熱（≤3倍）標註「波動」，不強制降級 | 仍有風險 |
| 同星座但>20° | 不計為合相 | 正常處理 | — |

### 7.3 與紫微斗數協同

若紫微斗數同時出現「太陽化忌」或「父母宮化忌」，則大熱警報再強化一級（大熱強制降級至RR，即使為寬鬆合相）。

### 7.4 實戰案例

2026-05-24 第8場（浪漫勇士）：太陽金牛4°，羅睺金牛26° → 寬鬆合相（差22°，略超20°邊界，不計）。若在20°內，則大熱浪漫勇士將被降級，但實際冠軍，故邊界設定合理。


## 八、班次權重與實力壓制協同（引用 `sop/sop_master.md`）

印度占星劇本判定必須與紫微斗數及階段0前置過濾器協同。以下為印度占星專屬補充：

### 8.1 班次權重（一級賽）

當賽事為 G1 或香港一級賽：
- 即使爆冷指數 ≥6，**冷門強度仍鎖定最高 5/10**。
- 輸出劇本定性不得為「冷門劇本」，最多為「混合劇本」。

### 8.2 實力壓制（超班馬）

當賽事中存在「評分≥全場平均+15分 且 近三仗全勝」的馬匹：
- 若印度占星中太陽與羅睺未合相（或僅寬鬆），則該馬無視冷門信號，強制列為正路首選。
- 若太陽與羅睺緊密合相，則實力壓制減半（該馬仍列主角，但需防意外）。
