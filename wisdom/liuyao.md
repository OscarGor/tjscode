## 一、心法總綱

六爻占卜為 TJSCODE **內環第三流派**，以三枚銅錢起卦，專司「即時占問」與「爻象吉凶」。若梅花易數從馬名中窺見天意定數，奇門遁甲從時空盤中揭示大格局，則六爻從**當下一念**中捕捉賽事結果的具體應期與成敗細節。

**最高心訣**：
> 子孫為馬，妻財為賞。  
> 官鬼為阻，兄弟為爭。  
> 世為我斷，應為賽場。  
> 動爻為機，旺相為上。

**《增刪卜易》核心精髓**：
> 「刪繁就簡，直奔吉凶」— 純以五行生剋、日月力量為準。日月為提綱，掌管生殺大權。動爻為玄機，神兆機於動。


## 二、起卦方法

### 2.1 工具與心法

以三枚相同銅錢（或硬幣），心念專注於所問之事（某匹馬能否勝出／整場賽事吉凶），擲六次，自下而上記錄爻象。

- 正面（字）為陰，反面（花）為陽（傳統設定，依實際工具而定，需統一）。
- 每次擲出，記錄：
  - 一個背面（花）：少陽 ⚊（不變）
  - 兩個背面：少陰 ⚋（不變）
  - 三個背面：老陽 ⚊（動爻，陽變陰）
  - 無背面（三個正面）：老陰 ⚋（動爻，陰變陽）

### 2.2 成卦

六次記錄由下往上排：
- 初爻（第一擲）、二爻、三爻、四爻、五爻、上爻（第六擲）。
- 有動爻（老陽/老陰）則生成變卦：陽爻變陰，陰爻變陽。
- 若無動爻，以本卦為準，不取變卦。

**記錄格式**：
本卦：䷔ 火雷噬嗑
變卦：䷕ 山火賁（初爻、四爻動）

## 三、用神體系

六爻以「用神」為核心，賽馬預測中特有以下用神對應：

### 3.1 主要用神

| 六親 | 代表 | 賽馬含義 |
|------|------|----------|
| **子孫爻** | **馬匹自身** | 活力、競爭力、狀態好壞。子孫旺相則馬勇。 |
| **妻財爻** | **獎金回報** | 該馬是否值得押注，回報是否可觀。 |
| **官鬼爻** | **阻礙、意外** | 犯規、傷病、意外、研訊。官鬼發動大凶。 |
| **兄弟爻** | **競爭對手** | 同場其他馬匹造成的威脅。 |
| **父母爻** | **外在條件** | 場地、天氣、騎師指令等外部因素。 |

### 3.2 世應關係

| 爻位 | 代表 | 賽馬含義 |
|------|------|----------|
| **世爻** | 我方（馬匹自身） | 以馬匹落宮天干為世爻（DP-039v2） |
| **應爻** | 對方（賽場環境） | 賽事整體環境，應爻生世則利馬匹發揮。 |


## 四、判斷核心公式

### 4.1 馬匹狀態判斷（子孫爻為主）

| 情況 | 判斷 | 部署建議 |
|------|------|----------|
| 子孫爻旺相 + 無沖剋 | 狀態上佳 | 優先考慮 |
| 子孫爻旺相 + 動爻來生 | 狀態大勇，有外力加持 | 可作核心部署 |
| 子孫爻休囚 + 官鬼發動 | 受制受阻，易有意外 | 降級或排除 |
| 子孫爻入墓 / 旬空 | 能量被封，虛有其表 | 謹慎，不宜重注 |
| 子孫爻月破 / 日破 | 當日狀態極差 | 強制排除 |
| 子孫爻發動化進神 | 狀態愈戰愈勇 | 後追馬加分 |
| 子孫爻發動化退神 | 狀態下滑，後勁不繼 | 前領馬高危 |

### 4.2 回報判斷（妻財爻為主）

| 情況 | 判斷 | 含義 |
|------|------|------|
| 妻財爻旺相 + 子孫爻動 | 回報可觀 | 值得押注，獨贏/位置均可 |
| 妻財爻休囚 + 兄弟爻旺 | 競爭激烈，派彩不高 | 熱門馬不值博 |
| 妻財爻發動剋世 | 看似有回報但不利我方 | 陷阱，不宜沾手 |
| 妻財爻生世爻 | 財來就我 | 此馬部署可獲利 |

### 4.3 阻礙判斷（官鬼爻為主）

| 情況 | 判斷 | 處理 |
|------|------|------|
| 官鬼爻發動 | 必有意外/阻礙 | 該馬強制標註，外環重點審查 |
| 官鬼爻剋子孫 | 直接打擊馬匹 | 強制降級 |
| 官鬼爻臨白虎 | 傷病風險極高 | 建議排除 |
| 官鬼爻臨玄武 | 隱藏問題，幕後異常 | 標註觀察 |

### 4.4 世應吉凶

| 關係 | 判斷 |
|------|------|
| 應爻生世爻 | 賽場合我方，預測準確率高 |
| 應爻剋世爻 | 賽場不利我方，容易誤判，須加倍謹慎 |
| 世應比和 | 平穩，按正常卦象解讀 |
| 世爻空亡 | 我方判斷力下降，不宜重注 |


## 五、爻位意義

六個爻位各有其時空意義，用之於賽馬，可對應賽事進程：

| 爻位 | 賽事階段 | 含義 |
|------|----------|------|
| **初爻** | 起步 / 出閘 | 起步反應、出閘順利與否 |
| **二爻** | 前段 / 對峙 | 前800米形勢，位置爭奪 |
| **三爻** | 中段 / 轉彎 | 中途變奏，轉彎技術 |
| **四爻** | 後段 / 加速區 | 開始發力，突圍時機 |
| **五爻** | 終段 / 衝刺 | 終點前角逐，王者之位 |
| **上爻** | 終點後 / 結果 | 最終名次，賽後狀況 |

**動爻在何位，代表該階段將出現重大變數。**

## 七、旬空與月破日破

### 7.1 旬空 (DP-044)

每旬有兩日為空亡。若用神（子孫爻、妻財爻等）落入旬空之爻，其力量為虛，待出旬（填空）方能發揮。賽馬預測中，若賽日正值爻之旬空，該爻代表的徵兆為虛幻，不可盡信。

| 旬 | 空亡地支 |
|------|----------|
| 甲子旬 | 戌亥空 |
| 甲戌旬 | 申酉空 |
| 甲申旬 | 午未空 |
| 甲午旬 | 辰巳空 |
| 甲辰旬 | 寅卯空 |
| 甲寅旬 | 子丑空 |

**旬空處理**：
- 旬空之爻臨日辰 → 出空，不降級
- 旬空+「胎」「養」長生 → 潛伏蓄力型，降級減半
- 旬空+月破同現 → 強制排除

### 7.2 月破日破

- **月破**：爻之五行被當月地支沖剋。代表整月不利，狀態持續低迷。
- **日破**：爻之五行被當日日支沖剋。代表當日狀態極差，即使卦象吉亦難發揮。

> ⚠️ 若子孫爻逢日破，該馬當日強制排除，不可因其他卦象吉而復活（憲法級規則）。

## 九、《增刪卜易》五層斷卦SOP (DP-039v2)

此為六爻預測學的最高實戰心法，純以五行生剋、日月力量為準。

### 第一層：立太極（定焦點）

- 以馬匹落宮天干為世爻（馬匹自身）
- 妻財爻 = 彩池收益用神
- 子孫爻 = 入圍福神
- 官鬼爻 = 競爭壓力
- 用神多現取動爻、臨日月之爻優先

### 第二層：審天時（定能量底色）

- 日辰對世爻生剋：臨日/日生=旺，日剋/日洩=衰
- 月建對世爻生剋：臨月/月生=旺，月剋/月洩=衰
- DP-044旬空月破即時判定：落宮逢旬空/月破 → 自動降一級稀有度
- 得日月雙生 → 加半星；受日月雙剋 → 減半星

### 第三層：辨動靜（看事情玄機）

- 梅花動爻為神兆機於動
- 動爻生世 = 原神（加分）
- 動爻剋世 = 忌神（減分）
- 變卦回頭生動爻 = 加分；回頭剋動爻 = 減分

### 第四層：評吉凶（下終極結論）

| 世爻 | 動爻 | 吉凶 |
|------|------|------|
| 旺相 | 原神生 | 大吉 |
| 旺相 | 忌神剋 | 受阻 |
| 衰弱 | 原神生 | 克處逢生 |
| 衰弱 | 忌神剋 | 大凶 |

- 克處逢生 → 名次可能受困後回升
- 貪合忘克 → 擠阻可避開

### 第五層：推應期（定名次區間）

| 世爻+子孫狀態 | 名次區間 |
|---------------|----------|
| 世旺+子旺 | 1-3名 |
| 世旺+子弱 | 2-4名 |
| 世弱+子旺 | 3-5名 |
| 世弱+子弱 | 6名或以後 |
| 旬空 | 末段追近，需留意後上跑法 |
| 月破 | 本場無望，下仗回升 |

## 十二、與其他流派的協作

- **梅花易數**：六爻為後天應期，梅花為先天定數。若梅花卦為體用大吉，六爻子孫爻又旺相發動，則天時地利人和皆備，可列為UR級核心馬。
- **奇門遁甲**：六爻的動爻位可與奇門的八門對應。例如六爻二爻動（前段）+ 奇門傷門落宮 = 前段競爭激烈，利搶位馬。
- **九星氣學**：六爻六獸中的青龍/白虎可對應九星的天沖/天柱，互相印證馬匹的爆發力或風險。騎馬配對吉凶可影響世爻能量判定。
- **鐵板神算**：鐵板騎練契合度吉 → 世爻加分；鐵板雙凶 → 世爻減分。
- **河洛數理**：世爻五行與落宮洛書數的關係可輔助判定旺衰。

- 本卦、變卦（如有動爻）

## 十五、歷史傳承註記

六爻占卜為中國最古老的決策工具之一。用之於賽馬，需摒除雜念，一心專注，方能感而遂通。本文獻繼承 GALLOPIA·玄 歷代六爻實戰經驗，每一條判斷公式皆經賽場驗證。學者當以誠心起卦，以邏輯解卦，不可因卦吉而妄喜，亦不可因卦凶而輕棄。
