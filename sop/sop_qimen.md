## 一、總綱

本 SOP 為奇門遁甲應用於賽馬預測的**強制操作標準**。每一匹參賽馬，不論賠率冷熱、不論檔位優劣，**必須完整通過以下十步分析**，不得跳步、不得合併、不得因個人經驗而簡化。

**核心原則**：一馬一盤，逐匹落宮。每一步皆有紀錄，每一步皆可稽查。

## 三、各步驟詳細操作指引

---

### 步驟一：落宮定位

**目的**：將馬匹編號轉化為奇門九宮位置，作為全部分析的地理基礎。

**操作規則**：
馬號 ÷ 8 = 商 ... 餘數
餘數 1 = 坎一宮（北方）
餘數 2 = 坤二宮（西南）
餘數 3 = 震三宮（東方）
餘數 4 = 巽四宮（東南）
餘數 5 = 中五宮（陽遁寄坤二、陰遁寄艮八）
餘數 6 = 乾六宮（西北）
餘數 7 = 兌七宮（西方）
餘數 0 = 艮八宮（東北）

text

**特別注意**：
- 餘數 5 必須註明陽遁或陰遁，並明確標示寄於何宮。
- 若馬號超過 8，以除 8 餘數為準，不可直接對應。

**輸出範例**：
> 馬號 3 → 3 ÷ 8 = 0...3 → 震三宮
> 馬號 13 → 13 ÷ 8 = 1...5 → 中五宮（陽遁寄坤二）

---

### 步驟二：四害檢查

**目的**：識別落宮是否受四害影響，四害為奇門大凶之象。

**四害定義**：

| 四害 | 定義 | 賽馬含義 |
|------|------|----------|
| 擊刑 | 天干受地盤相刑 | 騎師與場地不合，易犯規失位 |
| 干伏吟 | 天盤干與地盤干相同 | 馬匹狀態停滯，難以突破 |
| 門伏吟 | 八門落本宮 | 戰術無法施展，困守原地 |
| 星伏吟 | 九星落本宮 | 天時不利，能量被封鎖 |

**操作**：
1. 逐一檢查四害是否存在。
2. 若存在任何一害，須明確標示並記錄其賽馬含義。
3. 四害齊全者，該馬定為「凶格」，須特別標註。

**輸出範例**：
> 震三宮 - 四害檢查：
> - 擊刑：無
> - 干伏吟：無
> - 門伏吟：有（傷門落震宮）
> - 星伏吟：無
> 結論：有門伏吟，戰術施展受限。

---

### 步驟三：十二長生狀態

**目的**：判斷天盤干在落宮的旺衰狀態，決定馬匹當日「生命力」。

**十二長生順序**：
長生 → 沐浴 → 冠帶 → 臨官 → 帝旺 → 衰 → 病 → 死 → 墓 → 絕 → 胎 → 養

**五行長生位**：

| 天干 | 長生宮 | 帝旺宮 | 墓庫 |
|------|--------|--------|------|
| 甲 | 亥（乾六） | 卯（震三） | 未（坤二） |
| 乙 | 午（離九） | 寅（艮八） | 戌（乾六） |
| 丙 | 寅（艮八） | 午（離九） | 戌（乾六） |
| 丁 | 酉（兌七） | 巳（巽四） | 丑（艮八） |
| 戊 | 寅（艮八） | 午（離九） | 戌（乾六） |
| 己 | 酉（兌七） | 巳（巽四） | 丑（艮八） |
| 庚 | 巳（巽四） | 酉（兌七） | 丑（艮八） |
| 辛 | 子（坎一） | 申（坤二） | 辰（巽四） |
| 壬 | 申（坤二） | 子（坎一） | 辰（巽四） |
| 癸 | 卯（震三） | 亥（乾六） | 未（坤二） |

**操作**：
1. 確定天盤干及寄干。
2. 查表找出該干的長生位。
3. 順數至落宮，判定十二長生階段。
4. 長生/冠帶/臨官/帝旺 = 吉；衰/病/死/墓/絕 = 凶；胎/養 = 平。

**帝旺制化條款（DP-041v3）**：
- 帝旺+落宮五行與月令一致 → 真旺
- 帝旺+不一致 → 虛火
- 得制（外檔/負重/強敵）→ 加分半星
- 得洩（領放/早段用力）→ 加分半星
- 無制無洩 → 降半星
- 再生扶 → 過載降一星

**輸出範例**：
> 天盤干：甲 → 長生在乾六 → 落震三宮 = 帝旺（大吉）
> 月令夏季火 → 木休囚 → 虛火，需制化

### 步驟四：天干組合

**目的**：列出落宮所有天干，為步驟六的格局分析做準備。

**操作**：
1. 記錄天盤干。
2. 記錄地盤干。
3. 若中五宮寄宮，記錄寄干。
4. 所有天干逐一列出，不可遺漏。

**輸出範例**：
> 離九宮 - 天干組合：
> - 天盤干：丙
> - 地盤干：壬
> - 寄干：無

---

### 步驟五：地支組合

**目的**：列出落宮所有地支，為步驟六的格局分析做準備。

**操作**：
1. 記錄地盤地支。
2. 若天盤干自帶地支（甲子、乙丑等），一併記錄。
3. 若寄干帶入外地支，一併記錄。

**輸出範例**：
> 離九宮 - 地支組合：
> - 地盤地支：午
> - 天干自帶：丙午
> - 寄干地支：無

---

### 步驟六：天加地格局分析

**目的**：分析天盤干與地盤干的組合關係，此為奇門判斷吉凶的核心。

**操作規則**：
1. 以「天盤干 + 地盤干」形式組合。
2. 若有寄干，須分析「天盤干 + 寄干」兩組格局。
3. 常見吉格：青龍返首（戊+丙）、飛鳥跌穴（丙+戊）、玉女守門（丁+值使）、天乙會合（癸+戊）、太白入熒（庚+丙）等。
4. 常見凶格：青龍逃走（乙+辛）、白虎猖狂（辛+乙）、朱雀投江（丁+癸）、螣蛇夭矯（癸+丁）、地戶逢鬼（己+己）、日奇入墓（乙+己）等。
5. 無特殊格局者，以「平」標示。

**特殊格局速查表**：

| 格局 | 組合 | 吉凶 | 賽馬應用 |
|------|------|------|----------|
| 青龍返首 | 戊+丙 | 大吉 | 求財大旺，大熱門可信 |
| 飛鳥跌穴 | 丙+戊 | 大吉 | 機會主動送上門 |
| 天乙會合 | 癸+戊 | 吉（利客） | 暗中求財，貴人隱於暗處 |
| 太白入熒 | 庚+丙 | 吉（利客） | 後上突襲，冷門伏兵 |
| 熒入太白 | 丙+庚 | 吉（利主） | 防守反擊 |
| 青龍逃走 | 乙+辛 | 大凶 | 破財撤資，不可做膽 |
| 白虎猖狂 | 辛+乙 | 大凶 | 車禍血光，大熱倒灶 |
| 朱雀投江 | 丁+癸 | 大凶 | 文書遺失，合約失敗 |
| 螣蛇夭矯 | 癸+丁 | 凶 | 口舌官司，驚恐怪異 |
| 地戶逢鬼 | 己+己 | 凶 | 暗中消耗，百事不遂 |
| 日奇入墓 | 乙+己 | 平偏凶 | 希望被遮蔽，需吉門化解 |

**輸出範例**：
> 坎一宮 - 格局分析：
> - 癸+戊：天乙會合。暗中有貴人，利留後衝刺。評級：吉。
> - 無寄干。

### 步驟七：八門分析

**目的**：判斷八門吉凶及其對賽馬的影響。

**八門屬性**：

| 八門 | 吉凶 | 賽馬含義 |
|------|------|----------|
| 開門 | 大吉 | 開路先鋒，利領放、突破 |
| 休門 | 吉 | 養精蓄銳，利留前鬥後 |
| 生門 | 大吉 | 生機勃勃，利後上追趕 |
| 傷門 | 中（利競爭） | 主動出擊，利搶位、對抗 |
| 杜門 | 中（利隱藏） | 隱藏實力，利伏兵突襲 |
| 景門 | 中平 | 表面風光，宜睇定形勢 |
| 死門 | 大凶 | 前路不通，阻滯重重 |
| 驚門 | 凶 | 心神不寧，易犯錯失機 |

**門迫吉門降級（DP-075）**：
- 吉門（開、休、生）落入門迫宮位 → 不可做獨贏膽
- 開門門迫（落震三、巽四）→ 事業受阻
- 休門門迫（落離九）→ 懶散消極
- 生門門迫（落坎一）→ 財路斷絕

**操作**：
1. 記錄落宮八門。
2. 判定吉凶，並寫出賽馬含義。
3. 結合馬匹跑法特性：前領馬遇開門加分，後追馬遇生門加分。

**輸出範例**：
> 震三宮 - 八門：傷門。
> 含義：利競爭、利搶位、利主動出擊。符合該馬前領跑法，加分。

---

### 步驟八：八門+天盤組合分析

**目的**：將八門與天盤干組合，細化判斷。

**操作**：
1. 將落宮八門與所有天盤干（含寄干）配對。
2. 分析組合含義，例如：
   - 開門 + 戊 = 開門逢財，大利前領
   - 生門 + 丙 = 生生不息，後追強勁
   - 死門 + 庚 = 絕路逢阻，凶上加凶
   - 休門 + 太陰（八神） + 內檔 → 內檔相沖降級（DP-027v2）
3. 有寄干時須分析兩組。

**輸出範例**：
> 震三宮 - 八門+天盤組合：
> - 傷門 + 甲：木氣相疊，競爭力倍增（吉）
> - 無寄干。

---

### 步驟九：九星分析

**目的**：判斷天時能量對馬匹的加持或壓制。

**九星屬性**：

| 九星 | 五行 | 吉凶 | 賽馬含義 |
|------|------|------|----------|
| 天蓬 | 水 | 凶 | 冒進易失，利大膽突襲 |
| 天任 | 土 | 吉 | 穩健可靠，利均速跑法 |
| 天沖 | 木 | 中 | 衝擊力強，利短途爆發 |
| 天輔 | 木 | 吉 | 輔助得宜，利團隊配合 |
| 天英 | 火 | 中 | 光芒畢露，利焦點馬匹 |
| 天芮 | 土 | 凶 | 狀態不佳，宜避之（DP-033衰+天芮） |
| 天柱 | 金 | 凶 | 阻力重重，難以發揮 |
| 天心 | 金 | 吉 | 策劃得當，利戰術執行 |
| 天禽 | 土 | 大吉 | 居中調度，萬事皆宜 |

**星歸本位特別條款**：
- 九星落本宮 = 「星歸本位」，能量加倍。
- 例如：天沖星落震三宮（木歸木位），衝擊力倍增，短途馬大利。

**操作**：
1. 記錄落宮九星。
2. 判定吉凶及賽馬含義。
3. 檢查是否星歸本位，若是則特別標註並加重計分。

**輸出範例**：
> 震三宮 - 九星：天沖星。
> 含義：衝擊力強，利短途爆發。
> ⚠️ 星歸本位！天沖（木）落震三（木），能量加倍。評級：大吉。

### 步驟十：八神分析

**目的**：判斷神秘能量對馬匹的影響，八神為奇門最高層次能量。

**八神屬性**：

| 八神 | 吉凶 | 賽馬含義 |
|------|------|----------|
| 值符 | 大吉 | 天時所歸，貴人相助，頭馬之選 |
| 騰蛇 | 凶 | 虛浮不定，形勢反覆，易爆冷 |
| 太陰 | 吉 | 暗中策劃，伏兵突襲，冷門之兆 |
| 六合 | 吉 | 人馬合一，配合無間，連贏佳選 |
| 白虎 | 凶 | 剛猛過度，易傷己身，前領高危 |
| 玄武 | 凶 | 迷霧重重，形勢不明，宜避之 |
| 九地 | 中 | 穩守待機，不急不躁 |
| 九天 | 吉 | 氣勢如虹，主動出擊，大利放頭 |

**值符救應上限（DP-084）**：
- 奇門凶格（死門+壬己、傷門+伏宮等）逢值符
- 最多升一級（如 C→R，R→AR）
- 不可做獨贏膽，不可高於 SR

**操作**：
1. 記錄落宮八神。
2. 判定吉凶及賽馬含義。
3. 值符落宮之馬須特別標註，為「天選之馬」重要參考，但須遵守 DP-084 上限。

**輸出範例**：
> 震三宮 - 八神：值符。
> 含義：天時所歸，該駒獲神秘加持，列為頭馬候選。評級：大吉。

## 四、十步完成後的綜合評級

完成十步分析後，須對該馬匹進行綜合評級：

| 評級 | 標準 | 部署建議 |
|------|------|----------|
| **A級** | 過半步驟為吉，無重大凶格，若有星歸本位或值符加持 | 優先考慮入圍 |
| **B級** | 多數步驟為平或中吉，無致命凶格 | 可考慮入圍 |
| **C級** | 多數步驟為平，有一至兩個凶格 | 謹慎考慮，須外環驗證 |
| **D級** | 多數步驟為凶，或出現死門+凶格組合 | 排除候選（須雙舉證） |

## 七、強制執行條款

1. **覆蓋率 100%**：每場賽事全部參賽馬匹，必須逐匹完成十步分析，稽查部 (Tim) 負責核實。
2. **寄干雙重分析**：凡涉及中五宮寄干，所有天干相關步驟（步驟四、六、八）必須執行兩組分析。
3. **星歸本位標註**：步驟九必須明確標示是否星歸本位，漏標視為違規。
4. **值符強制標註**：步驟十若出現值符，須在綜合評級中特別列明，並遵守 DP-084 上限。
5. **特殊格局強制標註**：出現天乙會合、太白入熒、青龍逃走等格局，必須標註對應補丁。
6. **跳步處分**：任何跳步行為，稽查部有權退回整份分析，要求重做。
