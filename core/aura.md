## 一、總綱

能量氣場系統（Aura System）是 TJSCODE 的**動態能量追蹤模組**，用於捕捉馬匹、騎師及馬房在時間軸上的能量變化，補充卦象靜態分析的不足。

**核心原理**：
- 卦象揭示「天意所向」，氣場反映「當下狀態」。
- 卦象為體，氣場為用。卦象吉但氣場弱，須謹慎；卦象平但氣場強，可能為伏兵。

## 二、氣場分級系統

### 2.1 五級氣場

| 級別 | 名稱 | 能量描述 | 賽馬含義 |
|------|------|----------|----------|
| S | 巔峰 | 能量滿溢，氣勢如虹 | 勢不可擋，頭馬之選 |
| A | 強勢 | 能量充沛，狀態上佳 | 競爭力強，有力爭勝 |
| B | 平穩 | 能量穩定，無大波動 | 正常發揮，符合預期 |
| C | 低迷 | 能量下滑，隱現疲態 | 難以爭勝，僅望位置 |
| D | 衰竭 | 能量見底，氣場散亂 | 不宜考慮，強制排除 |

### 2.2 能量狀態標籤

在五級基礎上，附加狀態標籤以細化描述：

| 標籤 | 符號 | 含義 |
|------|------|------|
| 上升 | ↑ | 能量由低轉高，狀態回升中 |
| 下降 | ↓ | 能量由高轉低，狀態下滑中 |
| 穩定 | → | 能量持續平穩，無明顯變化 |
| 波動 | ⇵ | 能量忽高忽低，狀態不穩 |
| 爆發 | ↑↑ | 能量突然急升，可能為伏兵信號 |
| 急墜 | ↓↓ | 能量突然暴跌，強制警報 |

## 三、馬匹能量氣場分析

### 3.1 能量來源

馬匹氣場由以下因素綜合判定：

| 來源 | 權重 | 說明 |
|------|------|------|
| 近期賽績走勢 | 30% | 近三場名次及表現趨勢 |
| 體重變化 | 15% | 體重增減幅度（±15磅為警戒線） |
| 操練表現 | 15% | 晨操及試閘的時間及姿態 |
| 卦象能量 | 25% | 近期馬名卦及落宮卦的能量走勢 |
| 休息天數 | 15% | 久休復出或連戰之影響 |

### 3.2 能量計算公式
馬匹能量分數 = (賽績分 × 0.3) + (體重分 × 0.15) + (操練分 × 0.15) + (卦象分 × 0.25) + (休息分 × 0.15)

text

各項滿分為 10 分，總分對應五級氣場：

| 總分範圍 | 氣場級別 |
|----------|----------|
| 9.0 - 10.0 | S |
| 7.5 - 8.9 | A |
| 5.5 - 7.4 | B |
| 3.5 - 5.4 | C |
| 0.0 - 3.4 | D |

### 3.3 D級強制排除

任何馬匹若被評為 D 級（衰竭），不論內環卦象如何，**強制不納入最終部署**。此為外環降維之「形態降維」具體執行。

## 四、騎師能量氣場分析

### 4.1 能量來源

| 來源 | 權重 | 說明 |
|------|------|------|
| 近期頭馬數 | 35% | 近十場賽事的頭馬及上名率 |
| 九星氣學氣場 | 30% | Mika 以九星氣學判定騎師當日能量流向 |
| 騎師馬房配對 | 20% | 騎師與特定馬房合作的歷史成功率 |
| 當日賽事密度 | 15% | 當日座騎數量及場次分布 |

### 4.2 騎師能量指數

每名騎師在每個賽日獲派一個 1-10 分的能量指數。詳見 `data/jockey_index.md`。

**特別警報**：
- 能量指數低於 3 分 → 該騎師所有座騎在外環階段須強制審查。
- 能量指數 9 分或以上 → 該騎師座騎可獲內環評級上調一級（最多升至A級）。

## 五、馬房能量氣場分析

### 5.1 馬房能量週期

馬房（練馬師）的氣場呈週期性波動，分為以下階段：

| 階段 | 特徵 | 部署建議 |
|------|------|----------|
| 上升期 | 連續多場有馬入圍，士氣高昂 | 該馬房馬匹優先考慮 |
| 巔峰期 | 頻繁贏馬，氣場最強 | 馬房主力馬可重點關注 |
| 平台期 | 表現穩定，無大起落 | 正常分析，不特別加權 |
| 下滑期 | 頭馬減少，入圍率跌 | 該馬房馬匹謹慎處理 |
| 低谷期 | 長時間無馬贏，氣場低迷 | 馬房馬匹須降一級評級 |

### 5.2 馬房能量聯動

若同一馬房在同場派出多匹馬，須分析馬房能量是否足以支撐多線作戰：
- 馬房能量 S/A 級 → 多匹出戰可接受。
- 馬房能量 C/D 級 → 多匹出戰視為能量分散，各駒均須降級。

## 六、氣場與卦象的交互規則

| 卦象評級 | 氣場級別 | 最終調整 |
|----------|----------|----------|
| A | S/A | 維持A級，加分標註 |
| A | B | 維持A級，不作調整 |
| A | C | 降為B級，須外環審查 |
| A | D | 強制降為C級，可能排除 |
| B | S/A | 升為A級（伏兵信號） |
| B | B/C | 維持B級 |
| B | D | 降為C級 |
| C | S/A | 升為B級，標註為潛在伏兵 |
| C | B/C/D | 維持C級 |
| D | 任何級別 | 維持D級，強制排除 |

## 七、能量異常警報

以下情況出現時，須觸發能量異常警報，提交驗證部 (Angel) 深入追蹤：

1. **三日內能量急升↑↑**：馬匹或騎師在短時間內能量由 C 升至 A 或以上。
   - 可能原因：幕後部署轉向、秘密操練、馬匹狀態突破。
   - 處理：強制納入伏兵名單。

2. **三日內能量急墜↓↓**：馬匹或騎師能量由 A 急降至 C 或以下。
   - 可能原因：傷病隱患、幕後放棄、操練意外。
   - 處理：強制排除，除非卦象有極強直指信號。

3. **騎師能量與座騎卦象極端矛盾**：騎師能量 D 但座騎卦象 A。
   - 處理：提交戰略部 Ryan 審查，以卦象為優先但須雙舉證。
