## 一、總綱

本檔案記錄 TJSCODE 在實戰中對特定奇門遁甲格局的**驗證結果**與**規律發現**。每一條記錄皆源於真實賽事的成敗案例，是奇門遁甲從「理論格局」進化到「賽馬實戰應用」的核心數據庫。

**核心原則**：格局無絕對吉凶，落宮與月令決定成敗。同一格局在不同檔位、不同八神配合下，可呈現截然不同的賽馬結果。


## 二、天干格局驗證

### 2.1 青龍入海（癸 + 戊，坎一宮）

| 項目 | 內容 |
|------|------|
| **補丁** | DP-057v2 |
| **理論要義** | 龍入滄海，一展所長。戊土入坎水，實力與環境完美配合。 |
| **喜忌法則** | 喜風平浪靜（步速平穩、內檔、無碰撞）；忌驚濤駭浪（快步速、外檔、碰撞） |
| **判例一** | **嵐臣**（R2，坎一宮，癸+戊，值符，檔1內檔，步速平穩）：喜多於忌，季軍。 |
| **判例二** | **爽飆王子**（R2，同宮，無值符，檔10外檔，出閘笨拙）：忌多於喜，第8名。 |
| **教訓** | 同宮同格，但因檔位、八神、出閘情況不同而結果天壤之別。青龍入海非萬能，必須配合喜忌法則。 |

### 2.2 小蛇化龍（壬 + 戊）

| 項目 | 內容 |
|------|------|
| **理論要義** | 潛力蛻變，適合伏兵。壬水得戊土制而有序，小蛇化龍需條件配合。 |
| **判例一** | **志醒大將**（R1，坎一宮，壬+戊，休門+太陰，檔1）：化龍失敗，內檔受困，第四名。 |
| **判例二** | **極速之子**（R2，同宮，檔10）：外檔利迂迴，第5名。 |
| **教訓** | 小蛇化龍需要「水道」（外檔留後），內檔跟前則化龍受阻。 |

### 2.3 白虎出力（辛 + 庚）

| 項目 | 內容 |
|------|------|
| **理論要義** | 刀刃相交，拼搏有得。辛金剋庚金，競爭激烈，傷門同宮時碰撞風險高。 |
| **判例一** | **威力非凡**（R1，震三宮，辛+庚，傷門，值符，檔4）：值符化解擊刑，從外檔留後避開碰撞，末段突圍，冠軍。 |
| **判例二** | **幸福約定**（R2，震三宮，辛+庚，傷門，太陰，檔4）：擊刑起步擠迫收慢，第四名。 |
| **教訓** | 同宮同格，威力非凡因值符護身且領放型不受擊刑影響，幸福約定因需卦待時且非領放型而受損。 |

### 2.4 星隨月轉（丁 + 丙）

| 項目 | 內容 |
|------|------|
| **理論要義** | 貴人相助但不持久。丁火助丙火，但入墓或氣絕時力弱。 |
| **判例** | **手到再來**（R1，乾六宮，丁+丙，開門，4.5倍大熱，第13名）：丁入墓+氣絕，開門吉但力弱。 |
| **教訓** | 星隨月轉遇入墓則成虛格，不可因開門吉而高估。 |

### 2.5 青龍返首（戊 + 丙）

| 項目 | 內容 |
|------|------|
| **理論要義** | 龍回首，勢不可擋。大利前領馬，一放到底。 |
| **實戰狀態** | 兩場回測未出現此格，待未來驗證。 |

### 2.6 天網四張（癸 + 辛）

| 項目 | 內容 |
|------|------|
| **理論要義** | 困局難逃。癸水被辛金所困，杜門同宮時阻塞更甚。 |
| **判例** | **肥仔精神**（R1，巽四宮，癸+辛，杜門，8.9倍，第10名）：天網四張+杜門阻塞，困局應驗。 |
| **教訓** | 天網四張為困局信號，即使賠率非冷門亦難逃脫。 |

### 2.7 地戶逢鬼（己 + 己）

| 項目 | 內容 |
|------|------|
| **補丁** | DP-016 |
| **理論要義** | 伏吟格，穩定蓄力。在得地宮位（艮八宮）不凶。 |
| **判例一** | **神駒馬靈**（R1，艮八宮，己+己，死門得地，白虎，3.2倍，季軍）：伏吟得地，穩定入圍。 |
| **判例二** | **勁爽**（R1，同宮，18倍，亞軍）：同格，養字蓄力，伏兵成功。 |
| **教訓** | 伏吟格在得地宮位非凶反吉，是蓄力型的典型信號。 |


## 三、十二長生驗證

### 3.1 帝旺（真旺 vs 虛火）

| 項目 | 內容 |
|------|------|
| **補丁** | DP-041v3 |
| **理論要義** | 帝旺為巔峰極致，需制化。真旺（落宮五行與月令一致）+得制得洩=大吉；虛火（不一致）+無制=凶。 |
| **判例一** | **包裝明將**（R2，離九宮帝旺+巳月火=真旺，外檔14=官殺制衡）：得制，亞軍。 |
| **判例二** | **熊噹噹**（R1，離九宮帝旺+巳月火=真旺，內檔=無制無洩）：無制，第8名。 |
| **判例三** | **東方魅影**（R1，離九宮帝旺+巳月火=真旺，無制+卦象凶）：排除正確，第12名。 |
| **教訓** | 帝旺真旺必須有制化（外檔/領放/強敵）才能轉化為賽場優勢。無制則虛火，表面風光。 |

### 3.2 胎位（潛伏待機）

| 項目 | 內容 |
|------|------|
| **理論要義** | 胎為潛伏，能量未成熟，需強勢星門神配合才能化潛伏為突圍。 |
| **判例一** | **威力非凡**（R1，胎+值符+傷門）：值符護身，胎位化潛伏為末段突圍，冠軍。 |
| **判例二** | **志醒大將**（R1，胎+休門+太陰）：休門太陰陰柔，胎位未能化龍，第四名。 |
| **教訓** | 胎位的成敗取決於八神八門的配合。值符/白虎=突圍；休門/太陰=潛伏失敗。 |

### 3.3 養位（蓄力待發）

| 項目 | 內容 |
|------|------|
| **理論要義** | 養為蓄力，不顯山露水。黑馬潛質，冷門伏兵首選。 |
| **判例一** | **勁爽**（R1，養+死門得地+白虎）：蓄力成功，末段衝刺，亞軍。 |
| **判例二** | **神駒馬靈**（R1，養+死門得地+白虎）：蓄力穩定，季軍。 |
| **教訓** | 養位在艮八宮得地時，為最穩定的入圍信號之一。 |

### 3.4 衰位（回落）+ 天芮病符

| 項目 | 內容 |
|------|------|
| **補丁** | DP-033 |
| **理論要義** | 衰位本身已為回落，再遇天芮病符星，雙重負面。 |
| **判例** | **競駿傳奇**（R1，衰+天芮，58倍，第9名）：梅花泰卦雙吉，但衰+天芮拖累，大敗。 |
| **教訓** | 衰+天芮為DP-033的經典判例。即使梅花極強，此組合出現時必須降級。 |

### 3.5 死位（氣衰）

| 項目 | 內容 |
|------|------|
| **理論要義** | 氣勢衰絕，缺乏競爭力。 |
| **判例一** | **競駿天下**（R1，死，兌七宮，35倍，第11名）：氣衰，冷門復活後仍大敗。 |
| **判例二** | **肥仔精神**（R1，死，巽四宮，8.9倍，第10名）：雙死，大敗。 |
| **教訓** | 死位為強烈排除信號，即使賠率非冷門亦不宜沾手。 |

### 3.6 絕位（氣絕）

| 項目 | 內容 |
|------|------|
| **理論要義** | 氣絕，生機斷絕，強制排除信號。 |
| **判例** | **手到再來**（R1，絕，乾六宮，4.5倍，第13名）：大熱門氣絕，大敗。 |
| **教訓** | 絕位+大熱門=高危組合，不可因賠率而忽視。 |


## 四、八門八神驗證

|------|------|
| **補丁** | DP-016 |
| **理論要義** | 死門在艮八宮（土宮）為得地，凶而不凶，轉為穩守蓄力。 |
| **判例** | 勁爽、神駒馬靈（R1，艮八宮，死門得地）：亞軍及季軍。 |
| **教訓** | 死門得地是DP-016的經典案例，不可因「死」字而排除。 |

| **教訓** | 傷門+擊刑=DP-042的經典判例，起步必受阻。 |
