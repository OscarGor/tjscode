## 系統簡介

TJSCODE（天機解馬）是一套**基於七術合參的賽馬預測系統**，由甲己好玄學顧問公司運營，前身為 GALLOPIA·玄。經架構重組後，現以**九階段 SOP、雙環驗證、PTCG 稀有度評級**為核心，輸出紀律嚴明的 **$200 固定投注部署**。

**核心哲學**：先讀懂天時，再尋找主角。數據的盡頭是玄學，玄學的盡頭是劇本。

## 快速開始

### 1. 閱讀憲法
`CONSTITUTION.md` — 憲法級鐵律，不可違反。

### 2. 執行 SOP
`sop/sop_master.md` — 九階段總 SOP，所有賽事預測必須依序執行，零跳步。

### 3. 查閱補丁
`rules/patches.md` — **106 條補丁**（DP-001 至 DP-106），每一條都是用錢換來的教訓。

### 4. 接收賽事
提供賽事資訊（馬名、騎師、練馬師、檔位、賠率等），系統將自動執行完整九階段 SOP。
